AngularJS иЎЁеҚ•зӨәдҫӢ

еҗҚеӯ—пјҡ Bill е§“ж°Ҹпјҡ Gates йҮҚзҪ® form = {"firstName":"Bill","lastName":"Gates"} master = {"firstName":"Bill","lastName":"Gates"}

еә”з”ЁзЁӢеәҸд»Јз Ғ

<div ng-app="myApp" ng-controller="formCtrl">
  <form novalidate>
    еҗҚеӯ—пјҡ<br>
    <input type="text" ng-model="user.firstName"><br>
    е§“ж°Ҹпјҡ<br>
    <input type="text" ng-model="user.lastName">
    <br><br>
    <button ng-click="reset()">йҮҚзҪ®</button>
  </form>
  <p>form = {{user}}</p>
  <p>master = {{master}}</p>
</div>

<script>
var app = angular.module('myApp', []);
app.controller('formCtrl', function($scope) {
  $scope.master = {firstName: "Bill", lastName: "Gates"};
  $scope.reset = function() {
    $scope.user = angular.copy($scope.master);
  };
  $scope.reset();
});
</script>

Example Explained

ng-app жҢҮд»Өе®ҡд№ү AngularJS еә”з”ЁзЁӢеәҸгҖӮ

ng-controller жҢҮд»Өе®ҡд№үеә”з”ЁзЁӢеәҸжҺ§еҲ¶еҷЁгҖӮ

ng-model жҢҮд»Өе°ҶдёӨдёӘиҫ“е…Ҙе…ғзҙ з»‘е®ҡеҲ°жЁЎеһӢдёӯзҡ„ user еҜ№иұЎгҖӮ

formCtrl жҺ§еҲ¶еҷЁдёә master еҜ№иұЎи®ҫзҪ®еҲқе§ӢеҖјпјҢе№¶е®ҡд№ү reset() ж–№жі•гҖӮ

reset() ж–№жі•е°Ҷ user еҜ№иұЎи®ҫзҪ®дёәзӯүдәҺ master еҜ№иұЎгҖӮ

ng-click жҢҮд»Өд»…еңЁеҚ•еҮ»жҢүй’®ж—¶жүҚи°ғз”Ё reset() ж–№жі•гҖӮ

жӯӨеә”з”ЁзЁӢеәҸдёҚйңҖиҰҒ novalidate еұһжҖ§пјҢдҪҶйҖҡеёёжӮЁдјҡеңЁ AngularJS иЎЁеҚ•дёӯдҪҝз”Ёе®ғпјҢд»ҘиҰҶзӣ–ж ҮеҮҶ HTML5 йӘҢиҜҒгҖӮ
