ngRoute жЁЎеқ—её®еҠ©жӮЁзҡ„еә”з”ЁзЁӢеәҸжҲҗдёәеҚ•йЎөеә”з”ЁзЁӢеәҸгҖӮ

д»Җд№ҲжҳҜ AngularJS дёӯзҡ„и·Ҝз”ұпјҹ

еҰӮжһңжӮЁжғіеҜјиҲӘеҲ°еә”з”ЁзЁӢеәҸдёӯзҡ„дёҚеҗҢйЎөйқўпјҢдҪҶд№ҹеёҢжңӣеә”з”ЁзЁӢеәҸжҲҗдёә SPAпјҲеҚ•йЎөеә”з”ЁзЁӢеәҸпјүпјҢе№¶дё”жІЎжңүйЎөйқўйҮҚиҪҪпјҢеҲҷеҸҜд»ҘдҪҝз”Ё ngRoute жЁЎеқ—гҖӮ

ngRoute жЁЎеқ—е°ҶжӮЁзҡ„еә”з”ЁзЁӢеәҸи·Ҝз”ұеҲ°дёҚеҗҢзҡ„йЎөйқўпјҢиҖҢж— йңҖйҮҚж–°еҠ иҪҪж•ҙдёӘеә”з”ЁзЁӢеәҸгҖӮ

жҺ§еҲ¶еҷЁ

дҪҝз”Ё $routeProviderпјҢжӮЁиҝҳеҸҜд»ҘдёәжҜҸдёӘвҖңи§ҶеӣҫвҖқе®ҡд№үдёҖдёӘжҺ§еҲ¶еҷЁгҖӮ

var app = angular.module("myApp", ["ngRoute"]);
app.config(function($routeProvider) {
  $routeProvider
  .when("/", {
    templateUrl : "main.htm"

app.controller("londonCtrl", function ($scope) {
  $scope.msg = "I love London";
});
app.controller("parisCtrl", function ($scope) {
  $scope.msg = "I love Paris";
});

жӮЁиҝҳеҸҜд»ҘдҪҝз”Ё template еұһжҖ§пјҢе®ғе…Ғи®ёжӮЁзӣҙжҺҘеңЁеұһжҖ§еҖјдёӯзј–еҶҷ HTMLпјҢиҖҢдёҚжҳҜеј•з”ЁйЎөйқўгҖӮ

е®һдҫӢ

зј–еҶҷжЁЎжқҝпјҡ

var app = angular.module("myApp", ["ngRoute"]);
app.config(function($routeProvider) {
  $routeProvider
  .when("/", {
    template : "<h1>Main</h1><p>Click on the links to change this content</p>"
  })
  .when("/banana", {
    template : "<h1>Banana</h1><p>Bananas contain around 75% water.</p>"
  })
  .when("/tomato", {
    template : "<h1>Tomato</h1><p>Tomatoes contain around 95% water.</p>"
  });
});

дәІиҮӘиҜ•дёҖиҜ•

otherwise ж–№жі•

еңЁеүҚйқўзҡ„дҫӢеӯҗдёӯпјҢжҲ‘д»¬дҪҝз”ЁдәҶ $routeProvider зҡ„ when ж–№жі•гҖӮ

жӮЁиҝҳеҸҜд»ҘдҪҝз”Ё otherwise ж–№жі•пјҢеҪ“е…¶д»–жүҖжңүи·Ҝз”ұйғҪдёҚеҢ№й…Қж—¶пјҢе®ғе°ҶжҲҗдёәй»ҳи®Өи·Ҝз”ұгҖӮ

е®һдҫӢ

еҰӮжһңж—ўжІЎжңүзӮ№еҮ» "Banana" й“ҫжҺҘд№ҹжІЎжңүзӮ№еҮ» "Tomato" й“ҫжҺҘпјҢиҜ·е‘ҠиҜүд»–д»¬пјҡ

var app = angular.module("myApp", ["ngRoute"]);
app.config(function($routeProvider) {
  $routeProvider
  .when("/banana", {
    template : "<h1>Banana</h1><p>Bananas contain around 75% water.</p>"
  })
  .when("/tomato", {
    template : "<h1>Tomato</h1><p>Tomatoes contain around 95% water.</p>"
  })
  .otherwise({
    template : "<h1>None</h1><p>Nothing has been selected</p>"
  });
});
