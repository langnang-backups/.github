AngularJS е…Ғи®ёжӮЁеҹәдәҺж•°з»„жҲ–еҜ№иұЎдёӯзҡ„йЎ№зӣ®еҲӣе»әдёӢжӢүеҲ—иЎЁгҖӮ

дҪҝз”Ё ng-options еҲӣе»әдёӢжӢүжЎҶ

еҰӮжһңжӮЁжғіеңЁ AngularJS дёӯеҹәдәҺеҜ№иұЎжҲ–ж•°з»„еҲӣе»әдёӢжӢүеҲ—иЎЁпјҢеә”иҜҘдҪҝз”Ё ng-options жҢҮд»Өпјҡ

е®һдҫӢ

<div ng-app="myApp" ng-controller="myCtrl">

<select ng-model="selectedName" ng-options="x for x in names">
</select>

</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
  $scope.names = ["Emil", "Tobias", "Linus"];
});
</script>

дәІиҮӘиҜ•дёҖиҜ•

ng-options дёҺ ng-repeat

жӮЁд№ҹеҸҜд»ҘдҪҝз”Ё ng-repeat жҢҮд»ӨжқҘеҲӣе»әзӣёеҗҢзҡ„дёӢжӢүеҲ—иЎЁпјҡ

е®һдҫӢ

<select>
  <option ng-repeat="x in names">{{x}}</option>
</select>

дәІиҮӘиҜ•дёҖиҜ•

з”ұдәҺ ng-repeat жҢҮд»Өдёәж•°з»„дёӯзҡ„жҜҸдёӘйЎ№зӣ®йҮҚеӨҚдёҖж®ө HTML д»Јз ҒпјҢеӣ жӯӨе®ғеҸҜз”ЁдәҺеңЁдёӢжӢүеҲ—иЎЁдёӯеҲӣе»әйҖүйЎ№пјҢдҪҶжҳҜ ng-options жҢҮд»ӨжҳҜдё“й—ЁдёәдёӢжӢүеҲ—иЎЁеЎ«е……йҖүйЎ№иҖҢи®ҫи®Ўзҡ„гҖӮ

еә”иҜҘдҪҝз”Ёе“ӘдёҖдёӘпјҹ

жӮЁеҸҜд»ҘдҪҝз”Ё ng-repeat жҢҮд»Өе’Ң ng-options жҢҮд»Өпјҡ

еҒҮи®ҫжӮЁжңүдёҖдёӘеҜ№иұЎж•°з»„пјҡ

$scope.cars = [
  {model : "Ford Mustang", color : "red"},
  {model : "Fiat 500", color : "white"},
  {model : "Volvo XC90", color : "black"}
];

е®һдҫӢ

дҪҝз”Ё ng-repeatпјҡ

<select ng-model="selectedCar">
  <option ng-repeat="x in cars" value="{{x.model}}">{{x.model}}</option>
</select>

<h1>You selected: {{selectedCar}}</h1>

дәІиҮӘиҜ•дёҖиҜ•

еҪ“дҪҝз”ЁеҖјдҪңдёәеҜ№иұЎж—¶пјҢдҪҝз”Ё ng-value д»Јжӣҝ valueпјҡ

е®һдҫӢ

е°Ҷ ng-repeat з”ЁдҪңеҜ№иұЎпјҡ

<select ng-model="selectedCar">
  <option ng-repeat="x in cars" ng-value="{{x}}">{{x.model}}</option>
</select>

<h1>You selected a {{selectedCar.color}} {{selectedCar.model}}</h1>

дәІиҮӘиҜ•дёҖиҜ•

е®һдҫӢ

дҪҝз”Ё ng-optionsпјҡ

<select ng-model="selectedCar" ng-options="x.model for x in cars">
</select>

<h1>You selected: {{selectedCar.model}}</h1>
<p>Its color is: {{selectedCar.color}}</p>

дәІиҮӘиҜ•дёҖиҜ•

еҪ“жүҖйҖүеҖјдёәеҜ№иұЎж—¶пјҢе®ғеҸҜд»ҘеҢ…еҗ«жӣҙеӨҡдҝЎжҒҜпјҢе№¶дё”жӮЁзҡ„еә”з”ЁзЁӢеәҸеҸҜд»ҘжӣҙеҠ зҒөжҙ»гҖӮ

жҲ‘д»¬е°ҶеңЁжң¬ж•ҷзЁӢдёӯдҪҝз”Ё ng-options жҢҮд»ӨгҖӮ

дҪңдёәеҜ№иұЎзҡ„ж•°жҚ®жәҗ

еңЁеүҚйқўзҡ„зӨәдҫӢдёӯпјҢж•°жҚ®жәҗжҳҜж•°з»„пјҢдҪҶжҲ‘д»¬д№ҹеҸҜд»ҘдҪҝз”ЁеҜ№иұЎгҖӮ

еҒҮи®ҫжӮЁжңүдёҖдёӘеёҰжңүй”®еҖјеҜ№зҡ„еҜ№иұЎпјҡ

$scope.cars = {
  car01 : "Ford",
  car02 : "Fiat",
  car03 : "Volvo"
};

ng-options еұһжҖ§дёӯзҡ„иЎЁиҫҫејҸеҜ№дәҺеҜ№иұЎжқҘиҜҙз•ҘжңүдёҚеҗҢпјҡ

е®һдҫӢ

дҪҝз”ЁеҜ№иұЎдҪңдёәж•°жҚ®жәҗпјҢx д»ЈиЎЁй”®пјҢy д»ЈиЎЁеҖјпјҡ

<select ng-model="selectedCar" ng-options="x for (x, y) in cars">
</select>

<h1>You selected: {{selectedCar}}</h1>

дәІиҮӘиҜ•дёҖиҜ•

жүҖйҖүзҡ„еҖје°Ҷе§Ӣз»ҲжҳҜй”®еҖјеҜ№дёӯзҡ„еҖјгҖӮ

й”®еҖјеҜ№дёӯзҡ„еҖјд№ҹеҸҜд»ҘжҳҜеҜ№иұЎпјҡ

е®һдҫӢ

жүҖйҖүзҡ„еҖјд»Қз„¶е°ҶжҳҜй”®еҖјеҜ№дёӯзҡ„еҖјпјҢеҸӘжҳҜиҝҷж¬Ўе®ғжҳҜдёҖдёӘеҜ№иұЎпјҡ

$scope.cars = {
  car01 : {brand : "Ford", model : "Mustang", color : "red"},
  car02 : {brand : "Fiat", model : "500", color : "white"},
  car03 : {brand : "Volvo", model : "XC90", color : "black"}
};

дәІиҮӘиҜ•дёҖиҜ•

дёӢжӢүеҲ—иЎЁдёӯзҡ„йҖүйЎ№дёҚеҝ…жҳҜй”®еҖјеҜ№дёӯзҡ„й”®пјҢе®ғд№ҹеҸҜд»ҘжҳҜеҖјпјҢжҲ–иҖ…жҳҜеҖјеҜ№иұЎзҡ„еұһжҖ§пјҡ

е®һдҫӢ

<select ng-model="selectedCar" ng-options="y.brand for (x, y) in cars">
</select>
