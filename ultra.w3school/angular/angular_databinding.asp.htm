AngularJS дёӯзҡ„ж•°жҚ®з»‘е®ҡжҳҜжЁЎеһӢе’Ңи§Ҷеӣҫд№Ӣй—ҙзҡ„еҗҢжӯҘгҖӮ

ж•°жҚ®жЁЎеһӢ

AngularJS еә”з”ЁзЁӢеәҸйҖҡеёёжңүдёҖдёӘж•°жҚ®жЁЎеһӢгҖӮж•°жҚ®жЁЎеһӢжҳҜеҸҜз”ЁдәҺеә”з”ЁзЁӢеәҸзҡ„ж•°жҚ®зҡ„йӣҶеҗҲгҖӮ

е®һдҫӢ

var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
  $scope.firstname = "Bill";
  $scope.lastname = "Gates";
});

HTML и§Ҷеӣҫ

жҳҫзӨә AngularJS еә”з”ЁзЁӢеәҸзҡ„ HTML е®№еҷЁз§°дёәи§ҶеӣҫгҖӮ

и§ҶеӣҫеҸҜд»Ҙи®ҝй—®жЁЎеһӢпјҢе№¶дё”жңүеҮ з§Қж–№ејҸеҸҜд»ҘеңЁи§ҶеӣҫдёӯжҳҫзӨәжЁЎеһӢж•°жҚ®гҖӮ

жӮЁеҸҜд»ҘдҪҝз”Ё ng-bind жҢҮд»ӨпјҢе®ғдјҡе°Ҷе…ғзҙ зҡ„ innerHTML з»‘е®ҡеҲ°жҢҮе®ҡзҡ„жЁЎеһӢеұһжҖ§пјҡ

е®һдҫӢ

<p ng-bind="firstname"></p>

дәІиҮӘиҜ•дёҖиҜ•

жӮЁиҝҳеҸҜд»ҘдҪҝз”ЁеҸҢеӨ§жӢ¬еҸ· {{ }} жқҘжҳҫзӨәжЁЎеһӢдёӯзҡ„еҶ…е®№пјҡ

е®һдҫӢ

<p>First name: {{firstname}}</p>

дәІиҮӘиҜ•дёҖиҜ•

жҲ–иҖ…пјҢжӮЁеҸҜд»ҘеңЁ HTML жҺ§д»¶дёҠдҪҝз”Ё ng-model жҢҮд»Өе°ҶжЁЎеһӢз»‘е®ҡеҲ°и§ҶеӣҫгҖӮ

ng-model жҢҮд»Ө

дҪҝз”Ё ng-model жҢҮд»Өе°ҶжЁЎеһӢдёӯзҡ„ж•°жҚ®з»‘е®ҡеҲ° HTML жҺ§д»¶пјҲinput, select, textareaпјүзҡ„и§ҶеӣҫдёҠгҖӮ

е®һдҫӢ

<input ng-model="firstname">

дәІиҮӘиҜ•дёҖиҜ•

ng-model жҢҮд»ӨжҸҗдҫӣдәҶжЁЎеһӢе’Ңи§Ҷеӣҫд№Ӣй—ҙзҡ„еҸҢеҗ‘з»‘е®ҡгҖӮ

еҸҢеҗ‘з»‘е®ҡ

AngularJS дёӯзҡ„ж•°жҚ®з»‘е®ҡжҳҜжЁЎеһӢе’Ңи§Ҷеӣҫд№Ӣй—ҙзҡ„еҗҢжӯҘгҖӮ

еҪ“жЁЎеһӢдёӯзҡ„ж•°жҚ®еҸ‘з”ҹеҸҳеҢ–ж—¶пјҢи§ҶеӣҫдјҡеҸҚжҳ иҝҷз§ҚеҸҳеҢ–пјҢиҖҢеҪ“и§Ҷеӣҫдёӯзҡ„ж•°жҚ®еҸ‘з”ҹеҸҳеҢ–ж—¶пјҢжЁЎеһӢд№ҹдјҡжӣҙж–°гҖӮиҝҷжҳҜз«ӢеҚідё”иҮӘеҠЁеҸ‘з”ҹзҡ„пјҢд»ҺиҖҢзЎ®дҝқжЁЎеһӢе’Ңи§Ҷеӣҫе§Ӣз»Ҳеҫ—еҲ°жӣҙж–°гҖӮ

е®һдҫӢ

<div ng-app="myApp" ng-controller="myCtrl">
  Name: <input ng-model="firstname">
  <h1>{{firstname}}</h1>
</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
  $scope.firstname = "Bill";
  $scope.lastname = "Gates";
});
</script>

AngularJS жҺ§еҲ¶еҷЁ

AngularJS дёӯзҡ„еә”з”ЁзЁӢеәҸз”ұжҺ§еҲ¶еҷЁжҺ§еҲ¶гҖӮиҜ·еңЁ AngularJS жҺ§еҲ¶еҷЁдёҖз« дёӯйҳ…иҜ»жңүе…іжҺ§еҲ¶еҷЁзҡ„еҶ…е®№гҖӮ

з”ұдәҺжЁЎеһӢе’Ңи§Ҷеӣҫзҡ„еҚіж—¶еҗҢжӯҘпјҢжҺ§еҲ¶еҷЁеҸҜд»ҘдёҺи§Ҷеӣҫе®Ңе…ЁеҲҶзҰ»пјҢиҖҢеҸӘйңҖдё“жіЁдәҺжЁЎеһӢж•°жҚ®гҖӮз”ұдәҺ AngularJS дёӯзҡ„ж•°жҚ®з»‘е®ҡпјҢи§Ҷеӣҫе°ҶеҸҚжҳ жҺ§еҲ¶еҷЁдёӯжүҖеҒҡзҡ„д»»дҪ•жӣҙж”№гҖӮ

е®һдҫӢ

<div ng-app="myApp" ng-controller="myCtrl">
  <h1 ng-click="changeName()">{{firstname}}</h1>
</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
  $scope.firstname = "Bill";
  $scope.changeName = function() {
    $scope.firstname = "Nelly";
  }
});
</script>
