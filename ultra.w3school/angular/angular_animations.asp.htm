ngAnimate жЁЎеқ—дёҚдјҡдҪҝдҪ зҡ„ HTML е…ғзҙ еҠЁз”»еҢ–пјҢдҪҶжҳҜеҪ“ ngAnimate жіЁж„ҸеҲ°жҹҗдәӣдәӢд»¶пјҲеҰӮ HTML е…ғзҙ зҡ„йҡҗи—ҸжҲ–жҳҫзӨәпјүж—¶пјҢиҜҘе…ғзҙ дјҡиҺ·еҫ—дёҖдәӣйў„е®ҡд№үзҡ„зұ»пјҢиҝҷдәӣзұ»еҸҜз”ЁдәҺеҲ¶дҪңеҠЁз”»гҖӮ

дҪҝз”Ё CSS иҝӣиЎҢеҠЁз”»еӨ„зҗҶ

жҲ‘д»¬еҸҜд»ҘдҪҝз”Ё CSS иҝҮжёЎжҲ– CSS еҠЁз”»жқҘдёә HTML е…ғзҙ ж·»еҠ еҠЁз”»ж•ҲжһңгҖӮжң¬ж•ҷзЁӢе°Ҷеҗ‘дҪ еұ•зӨәиҝҷдёӨиҖ…гҖӮ

иҰҒдәҶи§Јжңүе…і CSS еҠЁз”»зҡ„жӣҙеӨҡдҝЎжҒҜпјҢиҜ·еӯҰд№ жҲ‘д»¬зҡ„ CSS иҝҮжёЎж•ҷзЁӢе’Ң CSS еҠЁз”»ж•ҷзЁӢгҖӮ

CSS иҝҮжёЎ

CSS иҝҮжёЎе…Ғи®ёдҪ еңЁз»ҷе®ҡзҡ„жҢҒз»ӯж—¶й—ҙеҶ…пјҢе°ҶдёҖдёӘ CSS еұһжҖ§еҖје№іж»‘ең°жӣҙж”№дёәеҸҰдёҖдёӘеҖјпјҡ

е®һдҫӢ

еҪ“ DIV е…ғзҙ иҺ·еҫ— .ng-hide зұ»ж—¶пјҢиҝҮжёЎе°ҶжҢҒз»ӯ 0.5 з§’пјҢй«ҳеәҰе°Ҷд»Һ 100px е№іж»‘иҝҮжёЎеҲ° 0пјҡ

<style>
div {
  transition: all linear 0.5s;
  background-color: lightblue;
  height: 100px;
}

.ng-hide {
  height: 0;
}
</style>

CSS еҠЁз”»

CSS еҠЁз”»е…Ғи®ёдҪ еңЁз»ҷе®ҡзҡ„жҢҒз»ӯж—¶й—ҙеҶ…пјҢе°ҶдёҖдёӘ CSS еұһжҖ§еҖје№іж»‘ең°жӣҙж”№дёәеҸҰдёҖдёӘеҖјпјҡ

е®һдҫӢ

еҪ“ DIV е…ғзҙ иҺ·еҫ— .ng-hide зұ»ж—¶пјҢе°ҶиҝҗиЎҢ myChange еҠЁз”»пјҢиҜҘеҠЁз”»е°Ҷй«ҳеәҰд»Һ 100px е№іж»‘иҝҮжёЎеҲ° 0пјҡ

<style>
@keyframes myChange {
  from {
    height: 100px;
  } to {
    height: 0;
  }
}

div {
  height: 100px;
  background-color: lightblue;
}

div.ng-hide {
  animation: 0.5s myChange;
}
</style>
