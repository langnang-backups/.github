AngularJS жҺ§еҲ¶еҷЁ

AngularJS еә”з”ЁзЁӢеәҸз”ұжҺ§еҲ¶еҷЁжҺ§еҲ¶гҖӮ

ng-controller жҢҮд»Өе®ҡд№үеә”з”ЁзЁӢеәҸжҺ§еҲ¶еҷЁгҖӮ

жҺ§еҲ¶еҷЁжҳҜз”ұж ҮеҮҶзҡ„ JavaScript еҜ№иұЎжһ„йҖ еҷЁеҲӣе»әзҡ„ JavaScript еҜ№иұЎгҖӮ

AngularJS е®һдҫӢ

<div ng-app="myApp" ng-controller="myCtrl">  
  
еҗҚеӯ—: <input type="text" ng-model="firstName"><br>  
е§“ж°Ҹ: <input type="text" ng-model="lastName"><br>  
<br>  
е…ЁеҗҚ: {{firstName + " " + lastName}}  
  
</div>  
  
<script>  
var app = angular.module('myApp', []);  
app.controller('myCtrl', function($scope) {  
  $scope.firstName = "Bill";  
  $scope.lastName = "Gates";  
});  
</script>

дәІиҮӘиҜ•дёҖиҜ•

еә”з”ЁзЁӢеәҸиҜҙжҳҺпјҡ

AngularJS еә”з”ЁзЁӢеәҸз”ұ ng-app="myApp" е®ҡд№үгҖӮеә”з”ЁзЁӢеәҸеңЁ <div> еҶ…иҝҗиЎҢгҖӮ

ng-controller="myCtrl" еұһжҖ§жҳҜдёҖдёӘ AngularJS жҢҮд»ӨгҖӮе®ғе®ҡд№үдәҶдёҖдёӘжҺ§еҲ¶еҷЁгҖӮ

myCtrl еҮҪж•°жҳҜдёҖдёӘ JavaScript еҮҪж•°гҖӮ

AngularJS е°ҶдҪҝз”Ё $scope еҜ№иұЎи°ғз”ЁжҺ§еҲ¶еҷЁгҖӮ

еңЁ AngularJS дёӯпјҢ$scope жҳҜеә”з”ЁзЁӢеәҸеҜ№иұЎпјҲеә”з”ЁзЁӢеәҸеҸҳйҮҸе’ҢеҮҪж•°зҡ„жүҖжңүиҖ…пјүгҖӮ

жҺ§еҲ¶еҷЁеңЁиҢғеӣҙпјҲfirstName е’Ң lastNameпјүдёӯеҲӣе»әдёӨдёӘеұһжҖ§пјҲеҸҳйҮҸпјүгҖӮ

ng-model жҢҮд»Өе°Ҷиҫ“е…Ҙеӯ—ж®өз»‘е®ҡеҲ°жҺ§еҲ¶еҷЁеұһжҖ§пјҲfirstName е’Ң lastNameпјүгҖӮ

жҺ§еҲ¶еҷЁж–№жі•

дёҠйқўзҡ„зӨәдҫӢжј”зӨәдәҶдёҖдёӘе…·жңүдёӨдёӘеұһжҖ§пјҡlastName е’Ң firstName зҡ„жҺ§еҲ¶еҷЁеҜ№иұЎгҖӮ

жҺ§еҲ¶еҷЁиҝҳеҸҜд»Ҙе…·жңүж–№жі•пјҲдҪңдёәеҮҪж•°зҡ„еҸҳйҮҸпјүпјҡ

AngularJS е®һдҫӢ

<div ng-app="myApp" ng-controller="personCtrl">  
  
еҗҚеӯ—: <input type="text" ng-model="firstName"><br>  
е§“ж°Ҹ: <input type="text" ng-model="lastName"><br>  
<br>  
е…ЁеҗҚ: {{fullName()}}  
  
</div>  
  
<script>  
var app = angular.module('myApp', []);  
app.controller('personCtrl', function($scope) {  
  $scope.firstName = "Bill";  
  $scope.lastName = "Gates";  
  $scope.fullName = function() {  
    return $scope.firstName + " " + $scope.lastName;  
  };  
});  
</script>
