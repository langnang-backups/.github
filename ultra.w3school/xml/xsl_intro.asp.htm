XSLT жҳҜдёҖз§Қз”ЁдәҺе°Ҷ XML ж–ҮжЎЈиҪ¬жҚўдёә XHTML ж–ҮжЎЈжҲ–е…¶д»– XML ж–ҮжЎЈзҡ„иҜӯиЁҖгҖӮ

XPath жҳҜдёҖз§Қз”ЁдәҺеңЁ XML ж–ҮжЎЈдёӯиҝӣиЎҢеҜјиҲӘзҡ„иҜӯиЁҖгҖӮ

еӯҰд№ д№ӢеүҚпјҢжӮЁйңҖиҰҒе…·еӨҮзҡ„еҹәзЎҖзҹҘиҜҶпјҡ

еңЁжӮЁз»§з»ӯеӯҰд№ д№ӢеүҚпјҢйңҖиҰҒеҜ№д»ҘдёӢзҹҘиҜҶжңүеҹәжң¬зҡ„дәҶи§Јпјҡ

  • HTML / XHTML
  • XML / XML е‘ҪеҗҚз©әй—ҙ
  • XPath

еҰӮжһңжӮЁеёҢжңӣйҰ–е…ҲеӯҰд№ иҝҷдәӣйЎ№зӣ®пјҢиҜ·еңЁжҲ‘д»¬зҡ„ йҰ–йЎө и®ҝй—®иҝҷдәӣж•ҷзЁӢгҖӮ

д»Җд№ҲжҳҜ XSLTпјҹ

  • XSLT дҪҝз”Ё XPath еңЁ XML ж–ҮжЎЈдёӯиҝӣиЎҢеҜјиҲӘгҖӮ

XSLT = XSL иҪ¬жҚў

XSLT жҳҜ XSL дёӯжңҖйҮҚиҰҒзҡ„йғЁеҲҶгҖӮ

XSLT з”ЁдәҺе°ҶдёҖз§Қ XML ж–ҮжЎЈиҪ¬жҚўдёәеҸҰеӨ–дёҖз§Қ XML ж–ҮжЎЈпјҢжҲ–иҖ…еҸҜиў«жөҸи§ҲеҷЁиҜҶеҲ«зҡ„е…¶д»–зұ»еһӢзҡ„ж–ҮжЎЈпјҢжҜ”еҰӮ HTML е’Ң XHTMLгҖӮйҖҡеёёпјҢXSLT жҳҜйҖҡиҝҮжҠҠжҜҸдёӘ XML е…ғзҙ иҪ¬жҚўдёә (X)HTML е…ғзҙ жқҘе®ҢжҲҗиҝҷйЎ№е·ҘдҪңзҡ„гҖӮ

йҖҡиҝҮ XSLTпјҢжӮЁеҸҜд»Ҙеҗ‘жҲ–иҖ…д»Һиҫ“еҮәж–Үд»¶ж·»еҠ жҲ–з§»йҷӨе…ғзҙ е’ҢеұһжҖ§гҖӮжӮЁд№ҹеҸҜйҮҚж–°жҺ’еҲ—е…ғзҙ пјҢжү§иЎҢжөӢиҜ•е№¶еҶіе®ҡйҡҗи—ҸжҲ–жҳҫзӨәе“ӘдёӘе…ғзҙ пјҢзӯүзӯүгҖӮ

жҸҸиҝ°иҪ¬еҢ–иҝҮзЁӢзҡ„дёҖз§ҚйҖҡеёёзҡ„иҜҙжі•жҳҜпјҢXSLT жҠҠ XML жәҗж ‘иҪ¬жҚўдёә XML з»“жһңж ‘гҖӮ

XSLT дҪҝз”Ё XPath

XSLT дҪҝз”Ё XPath еңЁ XML ж–ҮжЎЈдёӯжҹҘжүҫдҝЎжҒҜгҖӮXPath иў«з”ЁжқҘйҖҡиҝҮе…ғзҙ е’ҢеұһжҖ§еңЁ XML ж–ҮжЎЈдёӯиҝӣиЎҢеҜјиҲӘгҖӮ

еҰӮжһңжӮЁеёҢжңӣйҰ–е…ҲеӯҰд№  XPathпјҢиҜ·и®ҝй—®жҲ‘д»¬зҡ„ XPath ж•ҷзЁӢгҖӮ

е®ғеҰӮдҪ•е·ҘдҪңпјҹ

еңЁиҪ¬жҚўиҝҮзЁӢдёӯпјҢXSLT дҪҝз”Ё XPath жқҘе®ҡд№үжәҗж–ҮжЎЈдёӯеҸҜеҢ№й…ҚдёҖдёӘжҲ–еӨҡдёӘйў„е®ҡд№үжЁЎжқҝзҡ„йғЁеҲҶгҖӮдёҖж—ҰеҢ№й…Қиў«жүҫеҲ°пјҢXSLT е°ұдјҡжҠҠжәҗж–ҮжЎЈзҡ„еҢ№й…ҚйғЁеҲҶиҪ¬жҚўдёәз»“жһңж–ҮжЎЈгҖӮ
