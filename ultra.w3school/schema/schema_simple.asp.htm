дёҚиҝҮпјҢвҖңд»…еҢ…еҗ«ж–Үжң¬вҖқиҝҷдёӘйҷҗе®ҡеҚҙеҫҲе®№жҳ“йҖ жҲҗиҜҜи§ЈгҖӮж–Үжң¬жңүеҫҲеӨҡзұ»еһӢгҖӮе®ғеҸҜд»ҘжҳҜ XML Schema е®ҡд№үдёӯеҢ…жӢ¬зҡ„зұ»еһӢдёӯзҡ„дёҖз§ҚпјҲеёғе°”гҖҒеӯ—з¬ҰдёІгҖҒж•°жҚ®зӯүзӯүпјүпјҢжҲ–иҖ…е®ғд№ҹеҸҜд»ҘжҳҜжӮЁиҮӘиЎҢе®ҡд№үзҡ„е®ҡеҲ¶зұ»еһӢгҖӮ
