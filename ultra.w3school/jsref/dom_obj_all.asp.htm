Element жҺҘеҸЈиЎЁзӨә HTML е…ғзҙ гҖҒXMLе…ғзҙ жҲ–ж Үи®°гҖӮtagName еұһжҖ§жҢҮе®ҡдәҶе…ғзҙ зҡ„еҗҚз§°гҖӮDocument зҡ„ documentElement еұһжҖ§еј•з”ЁиҝҷдёӘж–ҮжЎЈзҡ„ж № Element еҜ№иұЎгҖӮHTMLDocument еҜ№иұЎзҡ„ body еұһжҖ§д№ҹзұ»дјјпјҢе®ғеј•з”ЁдәҶж–ҮжЎЈзҡ„ <body> е…ғзҙ гҖӮиҰҒеңЁдёҖдёӘ HTML ж–ҮжЎЈдёӯжүҫеҲ°жҢҮе®ҡеҗҚз§°зҡ„е…ғзҙ пјҢдҪҝз”Ё Document.getElementById()пјҲе№¶йҖҡиҝҮ id еұһжҖ§з»ҷиҜҘе…ғзҙ дёҖдёӘе”ҜдёҖзҡ„еҗҚз§°пјүгҖӮиҰҒйҖҡиҝҮж Үи®°еҗҚжқҘе®ҡдҪҚе…ғзҙ пјҢдҪҝз”Ё getElementsByTagName()пјҢиҝҷж—ўжҳҜ Element зҡ„ж–№жі•д№ҹжҳҜ Document зҡ„ж–№жі•гҖӮеңЁ HTML ж–ҮжЎЈдёӯпјҢд№ҹеҸҜд»ҘдҪҝз”Ёзұ»дјјзҡ„ HTMLDocument.getElementsByName() ж–№жі•жқҘж №жҚ®е…ғзҙ зҡ„ name еұһжҖ§жқҘжҹҘжүҫе…ғзҙ гҖӮжңҖеҗҺпјҢеҸҜд»Ҙз”Ё Document.createElement() ж–№жі•пјҢеҲӣе»әжҸ’е…Ҙж–ҮжЎЈзҡ„ж–° Element е…ғзҙ гҖӮ

addEventListener() ж–№жі•пјҲеҸҠе…¶зү№е®ҡдәҺ IE зҡ„жӣҝд»Јж–№жі• attachEvent() пјүжҸҗдҫӣдәҶеңЁиҜҘе…ғзҙ дёҠдёәзү№е®ҡзұ»еһӢзҡ„дәӢд»¶жіЁеҶҢдәӢд»¶еҸҘжҹ„еҮҪж•°зҡ„ж–№жі•гҖӮд»ҺжҠҖжңҜдёҠи®ІпјҢaddEventListener() гҖҒremoveEventListener() е’Ң dispatchEvent() йғҪжҳҜз”ұ2зә§ DOM Events и§„иҢғзҡ„ EventTarget жҺҘеҸЈе®ҡд№үзҡ„гҖӮжүҖжңүзҡ„ Element еҜ№иұЎйғҪе®һзҺ°дәҶ EventTargetгҖӮ

иҝҷдёӘжҺҘеҸЈзҡ„еҗ„з§Қе…¶д»–ж–№жі•жҸҗдҫӣдәҶеҜ№е…ғзҙ зҡ„еұһжҖ§зҡ„и®ҝй—®гҖӮеңЁ HTML ж–ҮжЎЈдёӯпјҲд»ҘеҸҠи®ёеӨҡ XML ж–ҮжЎЈдёӯпјүпјҢжүҖжңүеұһжҖ§йғҪжңүз®ҖеҚ•зҡ„еӯ—з¬ҰдёІеҖјпјҢе№¶дё”дҪ еҸҜд»ҘдҪҝз”Ёз®ҖеҚ•ж–№жі• getAttribute() е’Ң setAttribute() иҝӣиЎҢжүҖйңҖзҡ„д»»дҪ•еұһжҖ§ж“ҚдҪңгҖӮ

еҰӮжһңдҪ еңЁдҪҝз”Ё XML ж–ҮжЎЈпјҢе®ғеҸҜиғҪеҢ…еҗ«дәҶ Entity еҸӮиҖғйЎөдҪңдёәеұһжҖ§еҖјзҡ„дёҖйғЁеҲҶпјҢдҪ е°Ҷеҝ…йЎ»дҪҝз”Ё Attr еҜ№иұЎеҸҠе…¶иҠӮзӮ№зҡ„еӯҗж ‘гҖӮдҪ еҸҜй’ҲеҜ№дёҖдёӘеұһжҖ§дҪҝз”Ё getAttributeNode() е’Ң setAttributeNode() жқҘиҺ·еҸ–е’Ңи®ҫзҪ® Attr еҜ№иұЎпјҢжҲ–иҖ…еҸҜд»ҘеңЁ Node жҺҘеҸЈзҡ„ attributes[] ж•°з»„йҒҚеҺҶ Attr иҠӮзӮ№гҖӮеҰӮжһңжӮЁдҪҝз”ЁдәҶдёҖдёӘз”ЁеҲ° XML еҗҚеӯ—з©әй—ҙзҡ„ XML ж–ҮжЎЈпјҢйңҖиҰҒдҪҝз”ЁеҗҚеӯ—еёҰжңү "NS" зҡ„еҗ„з§Қж–№жі•гҖӮ

еңЁ 1 зә§ DOM и§„иҢғдёӯпјҢ normalize() ж–№жі•жҳҜ Element жҺҘеҸЈзҡ„дёҖйғЁеҲҶгҖӮеңЁ 2 зә§и§„иҢғдёӯпјҢnormalize() еҲҷжҳҜ Node жҺҘеҸЈзҡ„дёҖйғЁеҲҶгҖӮжүҖжңү Element иҠӮзӮ№з»§жүҝиҝҷдёӘж–№жі•е№¶дё”д»Қз„¶еҸҜд»ҘдҪҝз”Ёе®ғгҖӮ
