<canvas> е…ғзҙ е®ҡд№ү HTML йЎөйқўдёӯзҡ„дҪҚеӣҫеҢәеҹҹгҖӮ

Canvas API е…Ғи®ё JavaScript еңЁз”»еёғдёҠз»ҳеҲ¶еӣҫеҪўгҖӮ

Canvas API еҸҜд»Ҙз»ҳеҲ¶еҪўзҠ¶гҖҒзәҝжқЎгҖҒжӣІзәҝгҖҒж–№жЎҶгҖҒж–Үжң¬е’ҢеӣҫеғҸпјҢд»ҘеҸҠйўңиүІгҖҒж—ӢиҪ¬гҖҒйҖҸжҳҺеәҰе’Ңе…¶д»–еғҸзҙ ж“ҚдҪңгҖӮ

е°Ҷ Canvas ж·»еҠ еҲ° HTML

жӮЁеҸҜд»ҘдҪҝз”Ё <canvas> ж ҮзӯҫеңЁ HTML йЎөйқўдёӯзҡ„д»»ж„ҸдҪҚзҪ®ж·»еҠ  canvas е…ғзҙ пјҡ

е®һдҫӢ

<canvas id="myCanvas" width="300" height="150"></canvas>

дәІиҮӘиҜ•дёҖиҜ•

еҰӮдҪ•и®ҝй—® Canvas е…ғзҙ 

жӮЁеҸҜд»ҘдҪҝз”Ё HTML DOM ж–№жі• getElementById() и®ҝй—® <canvas> е…ғзҙ пјҡ

const myCanvas = document.getElementById("myCanvas");

еҰӮйңҖеңЁз”»еёғдёҠз»ҳеӣҫпјҢжӮЁйңҖиҰҒеҲӣе»ә 2D дёҠдёӢж–ҮеҜ№иұЎпјҡ

const ctx = myCanvas.getContext("2d");

з¬”и®°

HTML <canvas> е…ғзҙ жң¬иә«жІЎжңүз»ҳеӣҫеҠҹиғҪгҖӮ

жӮЁеҝ…йЎ»дҪҝз”Ё JavaScript жқҘз»ҳеҲ¶д»»дҪ•еӣҫеҪўгҖӮ

getContext() ж–№жі•иҝ”еӣһдёҖдёӘеёҰжңүз»ҳеӣҫе·Ҙе…·пјҲж–№жі•пјүзҡ„еҜ№иұЎгҖӮ

еңЁз”»еёғдёҠз»ҳеӣҫ

еҲӣе»ә 2D дёҠдёӢж–ҮеҜ№иұЎд№ӢеҗҺпјҢжӮЁе°ұеҸҜд»ҘеңЁз”»еёғдёҠз»ҳеӣҫдәҶгҖӮ

дёӢйқўзҡ„ fillRect() ж–№жі•з»ҳеҲ¶дәҶй»‘иүІзҹ©еҪўпјҢе…¶е·ҰдёҠи§’дҪҚдәҺдҪҚзҪ® 20,20гҖӮиҜҘзҹ©еҪўе®Ҫ 150 еғҸзҙ пјҢй«ҳ 100 еғҸзҙ пјҡ

е®һдҫӢ

const myCanvas = document.getElementById("myCanvas");
const ctx = myCanvas.getContext("2d");

ctx.fillRect(20, 20, 150, 100);

дәІиҮӘиҜ•дёҖиҜ•

дҪҝз”ЁйўңиүІ

fillStyle еұһжҖ§и®ҫзҪ®з»ҳеӣҫеҜ№иұЎзҡ„еЎ«е……йўңиүІпјҡ

е®һдҫӢ

const myCanvas = document.getElementById("myCanvas");
const ctx = myCanvas.getContext("2d");

ctx.fillStyle = "red";
ctx.fillRect(20, 20, 150, 100);

дәІиҮӘиҜ•дёҖиҜ•

жӮЁиҝҳеҸҜд»ҘдҪҝз”Ё document.createElement() ж–№жі•еҲӣе»әдёҖдёӘж–°зҡ„ <canvas> е…ғзҙ пјҢе№¶е°ҶиҜҘе…ғзҙ ж·»еҠ еҲ°зҺ°жңү HTML йЎөйқўпјҡ

е®һдҫӢ

const myCanvas = document.createElement("canvas");
document.body.appendChild(myCanvas);
const ctx = myCanvas.getContext("2d");

ctx.fillStyle = "red";
ctx.fillRect(20, 20, 150, 100);

дәІиҮӘиҜ•дёҖиҜ•

и·Ҝеҫ„

еңЁз”»еёғдёҠз»ҳеӣҫзҡ„еёёи§Ғж–№жі•жҳҜпјҡ

  1. ејҖе§Ӣи·Ҝеҫ„ - beginPath()
  2. з§»еҠЁеҲ°дёҖдёӘзӮ№ - moveTo()
  3. еңЁи·Ҝеҫ„дёӯз»ҳеҲ¶ - lineTo()
  4. з»ҳеҲ¶и·Ҝеҫ„ - stroke()

е®һдҫӢ

const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");

ctx.beginPath();
ctx.moveTo(20, 20);
ctx.lineTo(20, 100);
ctx.lineTo(70, 100);
ctx.stroke();

дәІиҮӘиҜ•дёҖиҜ•

йўңиүІгҖҒж ·ејҸе’ҢйҳҙеҪұ

еұһжҖ§ жҸҸиҝ°
fillStyle и®ҫзҪ®жҲ–иҝ”еӣһз”ЁдәҺеЎ«е……з»ҳеӣҫзҡ„йўңиүІгҖҒжёҗеҸҳжҲ–еӣҫжЎҲгҖӮ
strokeStyle и®ҫзҪ®жҲ–иҝ”еӣһз”ЁдәҺз¬”еҲ’зҡ„йўңиүІгҖҒжёҗеҸҳжҲ–еӣҫжЎҲгҖӮ
shadowColor и®ҫзҪ®жҲ–иҝ”еӣһз”ЁдәҺйҳҙеҪұзҡ„йўңиүІгҖӮ
shadowBlur и®ҫзҪ®жҲ–иҝ”еӣһйҳҙеҪұзҡ„жЁЎзіҠзә§еҲ«гҖӮ
shadowOffsetX и®ҫзҪ®жҲ–иҝ”еӣһйҳҙеҪұеҲ°еҪўзҠ¶зҡ„ж°ҙе№іи·қзҰ»гҖӮ
shadowOffsetY и®ҫзҪ®жҲ–иҝ”еӣһйҳҙеҪұеҲ°еҪўзҠ¶зҡ„еһӮзӣҙи·қзҰ»гҖӮ
ж–№жі• жҸҸиҝ°
createLinearGradient() еҲӣе»әзәҝжҖ§жёҗеҸҳпјҲз”ЁдәҺз”»еёғеҶ…е®№пјүгҖӮ
createPattern() еңЁжҢҮе®ҡж–№еҗ‘йҮҚеӨҚжҢҮе®ҡзҡ„е…ғзҙ гҖӮ
createRadialGradient() еҲӣе»әеҫ„еҗ‘/еңҶеҪўжёҗеҸҳпјҲз”ЁдәҺз”»еёғеҶ…е®№пјүгҖӮ
addColorStop() и§„е®ҡжёҗеҸҳеҜ№иұЎдёӯзҡ„йўңиүІе’ҢеҒңжӯўдҪҚзҪ®гҖӮ

зәҝжқЎж ·ејҸ

еұһжҖ§ жҸҸиҝ°
lineCap и®ҫзҪ®жҲ–иҝ”еӣһзәҝзҡ„з«Ҝзӣ–ж ·ејҸгҖӮ
lineJoin и®ҫзҪ®жҲ–иҝ”еӣһдёӨжқЎзәҝзӣёдәӨж—¶еҲӣе»әзҡ„и§’зҡ„зұ»еһӢгҖӮ
lineWidth и®ҫзҪ®жҲ–иҝ”еӣһеҪ“еүҚзәҝе®ҪгҖӮ
miterLimit и®ҫзҪ®жҲ–иҝ”еӣһжңҖеӨ§ж–ңжҺҘй•ҝеәҰгҖӮ

зҹ©еҪў

ж–№жі• жҸҸиҝ°
rect() еҲӣе»әзҹ©еҪўгҖӮ
fillRect() з»ҳеҲ¶вҖңеЎ«е……зҡ„вҖқзҹ©еҪўгҖӮ
strokeRect() з»ҳеҲ¶зҹ©еҪўпјҲж— еЎ«е……пјүгҖӮ
clearRect() жё…йҷӨз»ҷе®ҡзҹ©еҪўеҶ…зҡ„жҢҮе®ҡеғҸзҙ гҖӮ

и·Ҝеҫ„

ж–№жі• жҸҸиҝ°
fill() еЎ«е……еҪ“еүҚеӣҫеҪўпјҲи·Ҝеҫ„пјүгҖӮ
stroke() е®һйҷ…дёҠз»ҳеҲ¶жӮЁе®ҡд№үзҡ„и·Ҝеҫ„гҖӮ
beginPath() ејҖе§Ӣи·Ҝеҫ„пјҢжҲ–йҮҚзҪ®еҪ“еүҚи·Ҝеҫ„гҖӮ
moveTo() е°Ҷи·Ҝеҫ„з§»еҠЁеҲ°з”»еёғдёӯзҡ„жҢҮе®ҡзӮ№пјҢиҖҢдёҚеҲӣе»әзәҝгҖӮ
closePath() еҲӣе»әд»ҺеҪ“еүҚзӮ№иҝ”еӣһиө·зӮ№зҡ„и·Ҝеҫ„гҖӮ
lineTo() ж·»еҠ дёҖдёӘж–°зӮ№е№¶еҲӣе»әдёҖжқЎд»ҺиҜҘзӮ№еҲ°з”»еёғдёӯжңҖеҗҺдёҖдёӘжҢҮе®ҡзӮ№зҡ„зәҝгҖӮ
clip() д»ҺеҺҹе§Ӣз”»еёғдёӯеүӘиЈҒд»»ж„ҸеҪўзҠ¶е’ҢеӨ§е°Ҹзҡ„еҢәеҹҹгҖӮ
quadraticCurveTo() еҲӣе»әдәҢж¬ЎиҙқеЎһе°”жӣІзәҝгҖӮ
bezierCurveTo() еҲӣе»әдёүж¬ЎиҙқеЎһе°”жӣІзәҝгҖӮ
arc() еҲӣе»әеңҶеј§/жӣІзәҝпјҲз”ЁдәҺеҲӣе»әеңҶжҲ–еңҶзҡ„дёҖйғЁеҲҶпјүгҖӮ
arcTo() еңЁдёӨжқЎеҲҮзәҝд№Ӣй—ҙеҲӣе»әеңҶеј§/жӣІзәҝгҖӮ
isPointInPath() еҰӮжһңжҢҮе®ҡзӮ№еңЁеҪ“еүҚи·Ҝеҫ„дёӯпјҢеҲҷиҝ”еӣһ trueпјҢеҗҰеҲҷиҝ”еӣһ falseгҖӮ

ж–№жі• жҸҸиҝ°
