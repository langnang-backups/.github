ж–№жі• жҸҸиҝ° иҝ”еӣһзұ»еһӢ

findWithinHorizon() иҝ”еӣһжҢҮе®ҡеӯ—з¬Ұж•°иҢғеӣҙеҶ…еҢ№й…ҚжӯЈеҲҷиЎЁиҫҫејҸзҡ„еӯ—з¬ҰдёІгҖӮ String
hasNext() еҰӮжһңиҝҳеҸҜд»ҘжүҫеҲ°еҸҰдёҖдёӘ tokenпјҢеҲҷиҝ”еӣһ trueгҖӮ boolean
hasNextBoolean() еҰӮжһңдёӢдёҖдёӘ token иЎЁзӨәеёғе°”еҖјпјҢеҲҷиҝ”еӣһ trueгҖӮ boolean
hasNextByte() еҰӮжһңдёӢдёҖдёӘ token иЎЁзӨә byte еҖјпјҢеҲҷиҝ”еӣһ trueгҖӮ boolean
hasNextDouble() еҰӮжһңдёӢдёҖдёӘ token иЎЁзӨәж•°еӯ—пјҢеҲҷиҝ”еӣһ trueгҖӮ boolean
hasNextFloat() еҰӮжһңдёӢдёҖдёӘ token иЎЁзӨәж•°еӯ—пјҢеҲҷиҝ”еӣһ trueгҖӮ boolean
hasNextInt() еҰӮжһңдёӢдёҖдёӘ token иЎЁзӨә int еҖјпјҢеҲҷиҝ”еӣһ trueгҖӮ boolean
