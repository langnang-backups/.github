Java иҝҗз®—з¬Ұ

иҝҗз®—з¬Ұз”ЁдәҺеҜ№еҸҳйҮҸе’ҢеҖјжү§иЎҢж“ҚдҪңгҖӮ

еңЁдёӢйқўзҡ„дҫӢеӯҗдёӯпјҢжҲ‘д»¬дҪҝз”Ё + иҝҗз®—з¬Ұе°ҶдёӨдёӘеҖјзӣёеҠ пјҡ

е®һдҫӢ

int x = 100 + 50;

дәІиҮӘиҜ•дёҖиҜ•

е°Ҫз®Ў + иҝҗз®—з¬Ұз»Ҹеёёз”ЁдәҺе°ҶдёӨдёӘеҖјзӣёеҠ пјҢе°ұеғҸдёҠйқўзҡ„дҫӢеӯҗдёҖж ·пјҢдҪҶе®ғд№ҹеҸҜд»Ҙз”ЁдәҺе°ҶдёҖдёӘеҸҳйҮҸе’ҢдёҖдёӘеҖјзӣёеҠ пјҢжҲ–иҖ…е°ҶдёҖдёӘеҸҳйҮҸе’ҢеҸҰдёҖдёӘеҸҳйҮҸзӣёеҠ пјҡ

е®һдҫӢ

int sum1 = 100 + 50;        // 150 (100 + 50)
int sum2 = sum1 + 250;      // 400 (150 + 250)
int sum3 = sum2 + sum2;     // 800 (400 + 400)

дәІиҮӘиҜ•дёҖиҜ•

Java е°Ҷиҝҗз®—з¬ҰеҲҶдёәд»ҘдёӢеҮ з»„пјҡ

  • з®—жңҜиҝҗз®—з¬Ұ
  • иөӢеҖјиҝҗз®—з¬Ұ
  • жҜ”иҫғиҝҗз®—з¬Ұ

з®—жңҜиҝҗз®—з¬Ұ

з®—жңҜиҝҗз®—з¬Ұз”ЁдәҺжү§иЎҢеёёи§Ғзҡ„ж•°еӯҰиҝҗз®—гҖӮ

иҝҗз®—з¬Ұ еҗҚз§° жҸҸиҝ° дҫӢеӯҗ иҜ•дёҖиҜ•
+ еҠ  е°ҶдёӨдёӘеҖјзӣёеҠ  x + y иҜ•дёҖиҜ•
- еҮҸ д»ҺеҸҰдёҖдёӘеҖјдёӯеҮҸеҺ»дёҖдёӘеҖј x - y иҜ•дёҖиҜ•
* д№ҳ дёӨдёӘеҖјзӣёд№ҳ x * y иҜ•дёҖиҜ•
/ йҷӨ е°ҶдёҖдёӘеҖјйҷӨд»ҘеҸҰдёҖдёӘеҖј x / y иҜ•дёҖиҜ•
% еҸ–дҪҷ иҝ”еӣһйҷӨжі•дҪҷж•° x % y иҜ•дёҖиҜ•
++ йҖ’еўһ е°ҶеҸҳйҮҸзҡ„еҖјеҠ  1 ++x иҜ•дёҖиҜ•
-- йҖ’еҮҸ е°ҶеҸҳйҮҸзҡ„еҖјеҮҸ 1 --x иҜ•дёҖиҜ•

Java иөӢеҖјиҝҗз®—з¬Ұ

иөӢеҖјиҝҗз®—з¬Ұз”ЁдәҺдёәеҸҳйҮҸиөӢеҖјгҖӮ

еңЁдёӢдҫӢдёӯпјҢжҲ‘д»¬дҪҝз”ЁиөӢеҖјиҝҗз®—з¬Ұ (=) е°ҶеҖј 10 еҲҶй…Қз»ҷеҗҚдёә x зҡ„еҸҳйҮҸпјҡ

е®һдҫӢ

int x = 10;

дәІиҮӘиҜ•дёҖиҜ•

еҠ жі•иөӢеҖјиҝҗз®—з¬Ұ (+=) еҗ‘еҸҳйҮҸеҠ дёҖдёӘеҖјпјҡ

е®һдҫӢ

int x = 10;
x += 5;

дәІиҮӘиҜ•дёҖиҜ•

жүҖжңүиөӢеҖјиҝҗз®—з¬Ұзҡ„еҲ—иЎЁпјҡ

иҝҗз®—з¬Ұ дҫӢеӯҗ зӯүеҗҢдәҺ иҜ•дёҖиҜ•
= x = 5 x = 5 иҜ•дёҖиҜ•
+= x += 3 x = x + 3 иҜ•дёҖиҜ•
-= x -= 3 x = x - 3 иҜ•дёҖиҜ•
*= x *= 3 x = x * 3 иҜ•дёҖиҜ•
/= x /= 3 x = x / 3 иҜ•дёҖиҜ•
%= x %= 3 x = x % 3 иҜ•дёҖиҜ•
&= x &= 3 x = x & 3 иҜ•дёҖиҜ•
|= x |= 3 x = x | 3 иҜ•дёҖиҜ•
^= x ^= 3 x = x ^ 3 иҜ•дёҖиҜ•
>>= x >>= 3 x = x >> 3 иҜ•дёҖиҜ•
<<= x <<= 3 x = x << 3 иҜ•дёҖиҜ•

Java жҜ”иҫғиҝҗз®—з¬Ұ

жҜ”иҫғиҝҗз®—з¬Ұз”ЁдәҺжҜ”иҫғдёӨдёӘеҖјпјҲжҲ–еҸҳйҮҸпјүгҖӮиҝҷеңЁзј–зЁӢдёӯеҫҲйҮҚиҰҒпјҢеӣ дёәе®ғеҸҜд»Ҙеё®еҠ©жҲ‘д»¬жүҫеҲ°зӯ”жЎҲе№¶еҒҡеҮәеҶізӯ–гҖӮ

жҜ”иҫғзҡ„иҝ”еӣһеҖјдёә true жҲ– falseгҖӮиҝҷдәӣеҖјиў«з§°дёәеёғе°”еҖјпјҢжӮЁеҸҜд»ҘеңЁж•°жҚ®зұ»еһӢе’Ң If..Else з« иҠӮдёӯеӯҰеҲ°жӣҙеӨҡзӣёе…іеҶ…е®№гҖӮ

еңЁдёӢйқўзҡ„дҫӢеӯҗдёӯпјҢжҲ‘д»¬дҪҝз”ЁеӨ§дәҺиҝҗз®—з¬Ұ (>) жқҘеҲӨж–ӯ 5 жҳҜеҗҰеӨ§дәҺ 3пјҡ

е®һдҫӢ

int x = 5;
int y = 3;
System.out.println(x > y); // иҝ”еӣһ trueпјҢеӣ дёә 5 еӨ§дәҺ 3

дәІиҮӘиҜ•дёҖиҜ•

иҝҗз®—з¬Ұ еҗҚз§° зӯүеҗҢдәҺ иҜ•дёҖиҜ•
== зӯүдәҺ x == y иҜ•дёҖиҜ•
!= дёҚзӣёзӯү x != y иҜ•дёҖиҜ•
> еӨ§дәҺ x > y иҜ•дёҖиҜ•
< е°ҸдәҺ x < y иҜ•дёҖиҜ•
>= еӨ§дәҺжҲ–зӯүдәҺ x >= y иҜ•дёҖиҜ•
<= е°ҸдәҺжҲ–зӯүдәҺ x <= y иҜ•дёҖиҜ•

Java йҖ»иҫ‘иҝҗз®—з¬Ұ

жӮЁиҝҳеҸҜд»ҘдҪҝз”ЁйҖ»иҫ‘иҝҗз®—з¬ҰжөӢиҜ• true жҲ– false еҖјгҖӮ

йҖ»иҫ‘иҝҗз®—з¬Ұз”ЁдәҺзЎ®е®ҡеҸҳйҮҸжҲ–еҖјд№Ӣй—ҙзҡ„йҖ»иҫ‘пјҡ

иҝҗз®—з¬Ұ еҗҚз§° жҸҸиҝ° дҫӢеӯҗ иҜ•дёҖиҜ•
&& йҖ»иҫ‘е’Ң еҰӮжһңдёӨжқЎиҜӯеҸҘйғҪдёәзңҹпјҢеҲҷиҝ”еӣһ trueгҖӮ x < 5 && x < 10 иҜ•дёҖиҜ•
|| йҖ»иҫ‘жҲ– еҰӮжһңе…¶дёӯдёҖжқЎиҜӯеҸҘдёәзңҹпјҢеҲҷиҝ”еӣһ trueгҖӮ x < 5 || x < 4 иҜ•дёҖиҜ•
! йҖ»иҫ‘еҗҰ еҸҚиҪ¬з»“жһңгҖӮеҰӮжһңз»“жһңдёәзңҹпјҢеҲҷиҝ”еӣһ falseгҖӮ !(x < 5 && x < 10) иҜ•дёҖиҜ•
