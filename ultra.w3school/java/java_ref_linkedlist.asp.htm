LinkedList зҡ„жүҖжңүж–№жі•

д»ҘдёӢиЎЁж јеҲ—еҮәдәҶ LinkedList зҡ„жүҖжңүж–№жі•гҖӮ

йғЁеҲҶж–№жі•дҪҝз”Ё LinkedList дёӯе…ғзҙ зҡ„зұ»еһӢдҪңдёәеҸӮж•°жҲ–иҝ”еӣһеҖјгҖӮеңЁиЎЁж јдёӯпјҢиҝҷз§Қзұ»еһӢе°Ҷиў«иЎЁзӨәдёә TгҖӮ

ж–№жі• жҸҸиҝ° иҝ”еӣһзұ»еһӢ
add() еҗ‘еҲ—иЎЁдёӯж·»еҠ дёҖдёӘе…ғзҙ гҖӮ boolean|void
addAll() еҗ‘еҲ—иЎЁдёӯж·»еҠ дёҖдёӘйӣҶеҗҲзҡ„жүҖжңүе…ғзҙ гҖӮ boolean
addFirst() еңЁеҲ—иЎЁејҖеӨҙж·»еҠ дёҖдёӘе…ғзҙ гҖӮ void
addLast() еңЁеҲ—иЎЁжң«е°ҫж·»еҠ дёҖдёӘе…ғзҙ гҖӮ void
clear() д»ҺеҲ—иЎЁдёӯз§»йҷӨжүҖжңүе…ғзҙ гҖӮ void
clone() еҲӣе»ә LinkedList зҡ„еүҜжң¬гҖӮ
contains() жЈҖжҹҘеҲ—иЎЁдёӯжҳҜеҗҰеҢ…еҗ«жҹҗдёӘе…ғзҙ гҖӮ boolean
descendingIterator() иҝ”еӣһдёҖдёӘиҝӯд»ЈеҷЁпјҢд»ҘйҖҶеәҸйҒҚеҺҶеҲ—иЎЁдёӯзҡ„е…ғзҙ гҖӮ Iterator<T>
element() жЈҖзҙўеҲ—иЎЁдёӯзҡ„з¬¬дёҖдёӘе…ғзҙ гҖӮдёҺ getFirst() зұ»дјјгҖӮ T
forEach() еҜ№еҲ—иЎЁдёӯзҡ„жҜҸдёӘе…ғзҙ жү§иЎҢдёҖдёӘж“ҚдҪңгҖӮ void

getLast() иҝ”еӣһеҲ—иЎЁдёӯзҡ„жңҖеҗҺдёҖдёӘе…ғзҙ гҖӮ T

isEmpty() жЈҖжҹҘеҲ—иЎЁжҳҜеҗҰдёәз©әгҖӮ boolean
