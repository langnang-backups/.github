jQuery иҜӯжі•

jQuery иҜӯжі•жҳҜдёә HTML е…ғзҙ зҡ„йҖүеҸ–зј–еҲ¶зҡ„пјҢеҸҜд»ҘеҜ№е…ғзҙ жү§иЎҢжҹҗдәӣж“ҚдҪңгҖӮ

еҹәзЎҖиҜӯжі•жҳҜпјҡ$(selector).action()

  • зҫҺе…ғз¬ҰеҸ·е®ҡд№ү jQuery
  • йҖүжӢ©з¬ҰпјҲselectorпјүвҖңжҹҘиҜўвҖқе’ҢвҖңжҹҘжүҫвҖқ HTML е…ғзҙ 
  • jQuery зҡ„ action() жү§иЎҢеҜ№е…ғзҙ зҡ„ж“ҚдҪң

зӨәдҫӢ

$(this).hide() - йҡҗи—ҸеҪ“еүҚе…ғзҙ

$(".test").hide() - йҡҗи—ҸжүҖжңү class="test" зҡ„жүҖжңүе…ғзҙ 

$("#test").hide() - йҡҗи—ҸжүҖжңү id="test" зҡ„е…ғзҙ
