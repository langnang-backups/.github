Connection еҜ№иұЎ

ADO Connection еҜ№иұЎз”ЁдәҺеҲӣе»әдёҖдёӘеҲ°иҫҫжҹҗдёӘж•°жҚ®жәҗзҡ„ејҖж”ҫиҝһжҺҘгҖӮйҖҡиҝҮжӯӨиҝһжҺҘпјҢжӮЁеҸҜд»ҘеҜ№дёҖдёӘж•°жҚ®еә“иҝӣиЎҢи®ҝй—®е’Ңж“ҚдҪңгҖӮ

еҰӮжһңйңҖиҰҒеӨҡж¬Ўи®ҝй—®жҹҗдёӘж•°жҚ®еә“пјҢжӮЁеә”еҪ“дҪҝз”Ё Connection еҜ№иұЎжқҘе»әз«ӢдёҖдёӘиҝһжҺҘгҖӮжӮЁд№ҹеҸҜд»Ҙз»Ҹз”ұдёҖдёӘ Command жҲ– Recordset еҜ№иұЎдј йҖ’дёҖдёӘиҝһжҺҘеӯ—з¬ҰдёІжқҘеҲӣе»әжҹҗдёӘиҝһжҺҘгҖӮдёҚиҝҮпјҢжӯӨзұ»иҝһжҺҘд»…д»…йҖӮеҗҲдёҖж¬Ўе…·дҪ“зҡ„з®ҖеҚ•зҡ„жҹҘиҜўгҖӮ

Attributes и®ҫзҪ®жҲ–иҝ”еӣһ Connection еҜ№иұЎзҡ„еұһжҖ§гҖӮ
CommandTimeout жҢҮзӨәеңЁз»Ҳжӯўе°қиҜ•е’Ңдә§з”ҹй”ҷиҜҜд№ӢеүҚжү§иЎҢе‘Ҫд»Өжңҹй—ҙйңҖзӯүеҫ…зҡ„ж—¶й—ҙгҖӮ
ConnectionString и®ҫзҪ®жҲ–иҝ”еӣһз”ЁдәҺе»әз«ӢиҝһжҺҘж•°жҚ®жәҗзҡ„з»ҶиҠӮдҝЎжҒҜгҖӮ
ConnectionTimeout жҢҮзӨәеңЁз»Ҳжӯўе°қиҜ•е’Ңдә§з”ҹй”ҷиҜҜеүҚе»әз«ӢиҝһжҺҘжңҹй—ҙжүҖзӯүеҫ…зҡ„ж—¶й—ҙгҖӮ
CursorLocation и®ҫзҪ®жҲ–иҝ”еӣһжёёж ҮжңҚеҠЎзҡ„дҪҚзҪ®гҖӮ
DefaultDatabase жҢҮзӨә Connection еҜ№иұЎзҡ„й»ҳи®Өж•°жҚ®еә“гҖӮ
IsolationLevel жҢҮзӨә Connection еҜ№иұЎзҡ„йҡ”зҰ»зә§еҲ«гҖӮ
Mode и®ҫзҪ®жҲ–иҝ”еӣһ provider зҡ„и®ҝй—®жқғйҷҗгҖӮ
Provider и®ҫзҪ®жҲ–иҝ”еӣһ Connection еҜ№иұЎжҸҗдҫӣиҖ…зҡ„еҗҚз§°гҖӮ
State иҝ”еӣһдёҖдёӘжҸҸиҝ°иҝһжҺҘжҳҜжү“ејҖиҝҳжҳҜе…ій—ӯзҡ„еҖјгҖӮ
