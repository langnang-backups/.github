readonly еұһжҖ§дёҚйңҖиҰҒеҖјгҖӮе®ғзӯүеҗҢдәҺ readonly="readonly"гҖӮ

disabled еұһжҖ§

disabled еұһжҖ§и§„е®ҡиҫ“е…Ҙеӯ—ж®өжҳҜзҰҒз”Ёзҡ„гҖӮ

иў«зҰҒз”Ёзҡ„е…ғзҙ жҳҜдёҚеҸҜз”Ёе’ҢдёҚеҸҜзӮ№еҮ»зҡ„гҖӮ

иў«зҰҒз”Ёзҡ„е…ғзҙ дёҚдјҡиў«жҸҗдәӨгҖӮ

е®һдҫӢ

<form action="">
 First name:<br>
<input type="text" name="firstname" value="Bill" disabled>
<br>
 Last name:<br>
<input type="text" name="lastname">
</form>

дәІиҮӘиҜ•дёҖиҜ•

disabled еұһжҖ§дёҚйңҖиҰҒеҖјгҖӮе®ғзӯүеҗҢдәҺ disabled="disabled"гҖӮ

size еұһжҖ§

size еұһжҖ§и§„е®ҡиҫ“е…Ҙеӯ—ж®өзҡ„е°әеҜёпјҲд»Ҙеӯ—з¬Ұи®Ўпјүпјҡ

е®һдҫӢ

<form action="">
 First name:<br>
<input type="text" name="firstname" value="Bill" size="40">
<br>
 Last name:<br>
<input type="text" name="lastname">
</form>

дәІиҮӘиҜ•дёҖиҜ•

maxlength еұһжҖ§

maxlength еұһжҖ§и§„е®ҡиҫ“е…Ҙеӯ—ж®өе…Ғи®ёзҡ„жңҖеӨ§й•ҝеәҰпјҡ

е®һдҫӢ

<form action="">
 First name:<br>
<input type="text" name="firstname" maxlength="10">
<br>
 Last name:<br>
<input type="text" name="lastname">
</form>

дәІиҮӘиҜ•дёҖиҜ•

еҰӮи®ҫзҪ® maxlength еұһжҖ§пјҢеҲҷиҫ“е…ҘжҺ§д»¶дёҚдјҡжҺҘеҸ—и¶…иҝҮжүҖе…Ғи®ёж•°зҡ„еӯ—з¬ҰгҖӮ

иҜҘеұһжҖ§дёҚдјҡжҸҗдҫӣд»»дҪ•еҸҚйҰҲгҖӮеҰӮжһңйңҖиҰҒжҸҗйҶ’з”ЁжҲ·пјҢеҲҷеҝ…йЎ»зј–еҶҷ JavaScript д»Јз ҒгҖӮ

жіЁйҮҠпјҡиҫ“е…ҘйҷҗеҲ¶е№¶йқһдёҮж— дёҖеӨұгҖӮJavaScript жҸҗдҫӣдәҶеҫҲеӨҡж–№жі•жқҘеўһеҠ йқһжі•иҫ“е…ҘгҖӮеҰӮйңҖе®үе…Ёең°йҷҗеҲ¶иҫ“е…ҘпјҢеҲҷжҺҘеҸ—иҖ…пјҲжңҚеҠЎеҷЁпјүеҝ…йЎ»еҗҢж—¶еҜ№йҷҗеҲ¶иҝӣиЎҢжЈҖжҹҘгҖӮ

autocomplete еұһжҖ§

autocomplete еұһжҖ§и§„е®ҡиЎЁеҚ•жҲ–иҫ“е…Ҙеӯ—ж®өжҳҜеҗҰеә”иҜҘиҮӘеҠЁе®ҢжҲҗгҖӮ

еҪ“иҮӘеҠЁе®ҢжҲҗејҖеҗҜпјҢжөҸи§ҲеҷЁдјҡеҹәдәҺз”ЁжҲ·д№ӢеүҚзҡ„иҫ“е…ҘеҖјиҮӘеҠЁеЎ«еҶҷеҖјгҖӮ

жҸҗзӨәпјҡжӮЁеҸҜд»ҘжҠҠиЎЁеҚ•зҡ„ autocomplete и®ҫзҪ®дёә onпјҢеҗҢж—¶жҠҠзү№е®ҡзҡ„иҫ“е…Ҙеӯ—ж®өи®ҫзҪ®дёә offпјҢеҸҚд№ӢдәҰз„¶гҖӮ

autocomplete еұһжҖ§йҖӮз”ЁдәҺ <form> д»ҘеҸҠеҰӮдёӢ <input> зұ»еһӢпјҡtextгҖҒsearchгҖҒurlгҖҒtelгҖҒemailгҖҒpasswordгҖҒdatepickersгҖҒrange д»ҘеҸҠ colorгҖӮ

е®һдҫӢ

иҮӘеҠЁе®ҢжҲҗејҖеҗҜзҡ„ HTML иЎЁеҚ•пјҲжҹҗдёӘиҫ“е…Ҙеӯ—ж®өдёә offпјүпјҡ

<form action="action_page.php" autocomplete="on">
   First name:<input type="text" name="fname"><br>
   Last name: <input type="text" name="lname"><br>
   E-mail: <input type="email" name="email" autocomplete="off"><br>
   <input type="submit">
</form>

дәІиҮӘиҜ•дёҖиҜ•

жҸҗзӨәпјҡеңЁжҹҗдәӣжөҸи§ҲеҷЁдёӯпјҢжӮЁд№ҹи®ёйңҖиҰҒжүӢеҠЁеҗҜз”ЁиҮӘеҠЁе®ҢжҲҗеҠҹиғҪгҖӮ

novalidate еұһжҖ§

novalidate еұһжҖ§еұһдәҺ <form> еұһжҖ§гҖӮ

еҰӮжһңи®ҫзҪ®пјҢеҲҷ novalidate и§„е®ҡеңЁжҸҗдәӨиЎЁеҚ•ж—¶дёҚеҜ№иЎЁеҚ•ж•°жҚ®иҝӣиЎҢйӘҢиҜҒгҖӮ

е®һдҫӢ

жҢҮзӨәиЎЁеҚ•еңЁиў«жҸҗдәӨж—¶дёҚиҝӣиЎҢйӘҢиҜҒпјҡ

<form action="action_page.php" novalidate>
   E-mail: <input type="email" name="user_email">
   <input type="submit">
</form>

дәІиҮӘиҜ•дёҖиҜ•

autofocus еұһжҖ§

autofocus еұһжҖ§жҳҜеёғе°”еұһжҖ§гҖӮ

еҰӮжһңи®ҫзҪ®пјҢеҲҷи§„е®ҡеҪ“йЎөйқўеҠ иҪҪж—¶ <input> е…ғзҙ еә”иҜҘиҮӘеҠЁиҺ·еҫ—з„ҰзӮ№гҖӮ

е®һдҫӢ

дҪҝ "First name" иҫ“е…Ҙеӯ—ж®өеңЁйЎөйқўеҠ иҪҪж—¶иҮӘеҠЁиҺ·еҫ—з„ҰзӮ№пјҡ

First name:<input type="text" name="fname" autofocus>

дәІиҮӘиҜ•дёҖиҜ•

form еұһжҖ§

form еұһжҖ§и§„е®ҡ <input> е…ғзҙ жүҖеұһзҡ„дёҖдёӘжҲ–еӨҡдёӘиЎЁеҚ•гҖӮ

жҸҗзӨәпјҡеҰӮйңҖеј•з”ЁдёҖдёӘд»ҘдёҠзҡ„иЎЁеҚ•пјҢиҜ·дҪҝз”Ёз©әж јеҲҶйҡ”зҡ„иЎЁеҚ• id еҲ—иЎЁгҖӮ

е®һдҫӢ

иҫ“е…Ҙеӯ—ж®өдҪҚдәҺ HTML иЎЁеҚ•д№ӢеӨ–пјҲдҪҶд»ҚеұһиЎЁеҚ•пјүпјҡ

<form action="action_page.php" id="form1">
   First name: <input type="text" name="fname"><br>
   <input type="submit" value="Submit">
</form>

 Last name: <input type="text" name="lname" form="form1">

дәІиҮӘиҜ•дёҖиҜ•

formaction еұһжҖ§

formaction еұһжҖ§и§„е®ҡеҪ“жҸҗдәӨиЎЁеҚ•ж—¶еӨ„зҗҶиҜҘиҫ“е…ҘжҺ§д»¶зҡ„ж–Үд»¶зҡ„ URLгҖӮ

formaction еұһжҖ§иҰҶзӣ– <form> е…ғзҙ зҡ„ action еұһжҖ§гҖӮ

formaction еұһжҖ§йҖӮз”ЁдәҺ type="submit" д»ҘеҸҠ type="image"гҖӮ

е®һдҫӢ

жӢҘжңүдёӨдёӘдёӨдёӘжҸҗдәӨжҢүй’®е№¶еҜ№дәҺдёҚеҗҢеҠЁдҪңзҡ„ HTML иЎЁеҚ•пјҡ

<form action="action_page.php">
   First name: <input type="text" name="fname"><br>
   Last name: <input type="text" name="lname"><br>
   <input type="submit" value="Submit"><br>
   <input type="submit" formaction="demo_admin.asp"
   value="Submit as admin">
</form>

дәІиҮӘиҜ•дёҖиҜ•

formenctype еұһжҖ§

formenctype еұһжҖ§и§„е®ҡеҪ“жҠҠиЎЁеҚ•ж•°жҚ®пјҲform-dataпјүжҸҗдәӨиҮіжңҚеҠЎеҷЁж—¶еҰӮдҪ•еҜ№е…¶иҝӣиЎҢзј–з ҒпјҲд»…й’ҲеҜ№ method="post" зҡ„иЎЁеҚ•пјүгҖӮ

formenctype еұһжҖ§иҰҶзӣ– <form> е…ғзҙ зҡ„ enctype еұһжҖ§гҖӮ

formenctype еұһжҖ§йҖӮз”ЁдәҺ type="submit" д»ҘеҸҠ type="image"гҖӮ

е®һдҫӢ

еҸ‘йҖҒй»ҳи®Өзј–з Ғд»ҘеҸҠзј–з Ғдёә "multipart/form-data"пјҲз¬¬дәҢдёӘжҸҗдәӨжҢүй’®пјүзҡ„иЎЁеҚ•ж•°жҚ®пјҲform-dataпјүпјҡ

<form action="demo_post_enctype.asp" method="post">
   First name: <input type="text" name="fname"><br>
   <input type="submit" value="Submit">
   <input type="submit" formenctype="multipart/form-data"
   value="Submit as Multipart/form-data">
</form>

дәІиҮӘиҜ•дёҖиҜ•

formmethod еұһжҖ§

formmethod еұһжҖ§е®ҡд№үз”Ёд»Ҙеҗ‘ action URL еҸ‘йҖҒиЎЁеҚ•ж•°жҚ®пјҲform-dataпјүзҡ„ HTTP ж–№жі•гҖӮ

formmethod еұһжҖ§иҰҶзӣ– <form> е…ғзҙ зҡ„ method еұһжҖ§гҖӮ

formmethod еұһжҖ§йҖӮз”ЁдәҺ type="submit" д»ҘеҸҠ type="image"гҖӮ

е®һдҫӢ

з¬¬дәҢдёӘжҸҗдәӨжҢүй’®иҰҶзӣ–иЎЁеҚ•зҡ„ HTTP ж–№жі•пјҡ

<form action="action_page.php" method="get">
   First name: <input type="text" name="fname"><br>
   Last name: <input type="text" name="lname"><br>
   <input type="submit" value="Submit">
   <input type="submit" formmethod="post" formaction="demo_post.asp"
   value="Submit using POST">
</form>

дәІиҮӘиҜ•дёҖиҜ•

formnovalidate еұһжҖ§

novalidate еұһжҖ§жҳҜеёғе°”еұһжҖ§гҖӮ

еҰӮжһңи®ҫзҪ®пјҢеҲҷи§„е®ҡеңЁжҸҗдәӨиЎЁеҚ•ж—¶дёҚеҜ№ <input> е…ғзҙ иҝӣиЎҢйӘҢиҜҒгҖӮ

formnovalidate еұһжҖ§иҰҶзӣ– <form> е…ғзҙ зҡ„ novalidate еұһжҖ§гҖӮ

formnovalidate еұһжҖ§еҸҜз”ЁдәҺ type="submit"гҖӮ

е®һдҫӢ

жӢҘжңүдёӨдёӘжҸҗдәӨжҢүй’®зҡ„иЎЁеҚ•пјҲйӘҢиҜҒе’ҢдёҚйӘҢиҜҒпјүпјҡ

<form action="action_page.php">
   E-mail: <input type="email" name="userid"><br>
   <input type="submit" value="Submit"><br>
   <input type="submit" formnovalidate value="Submit without validation">
</form>

дәІиҮӘиҜ•дёҖиҜ•

formtarget еұһжҖ§

formtarget еұһжҖ§и§„е®ҡзҡ„еҗҚз§°жҲ–е…ій”®иҜҚжҢҮзӨәжҸҗдәӨиЎЁеҚ•еҗҺеңЁдҪ•еӨ„жҳҫзӨәжҺҘж”¶еҲ°зҡ„е“Қеә”гҖӮ

formtarget еұһжҖ§дјҡиҰҶзӣ– <form> е…ғзҙ зҡ„ target еұһжҖ§гҖӮ

formtarget еұһжҖ§еҸҜдёҺ type="submit" е’Ң type="image" дҪҝз”ЁгҖӮ

е®һдҫӢ

иҝҷдёӘиЎЁеҚ•жңүдёӨдёӘжҸҗдәӨжҢүй’®пјҢеҜ№еә”дёҚеҗҢзҡ„зӣ®ж ҮзӘ—еҸЈпјҡ

<form action="action_page.php">
   First name: <input type="text" name="fname"><br>
   Last name: <input type="text" name="lname"><br>
   <input type="submit" value="Submit as normal">
   <input type="submit" formtarget="_blank"
   value="Submit to a new window">
</form>

height е’Ң width еұһжҖ§д»…з”ЁдәҺ <input type="image">гҖӮ

жіЁйҮҠпјҡиҜ·е§Ӣз»Ҳи§„е®ҡеӣҫеғҸзҡ„е°әеҜёгҖӮеҰӮжһңжөҸи§ҲеҷЁдёҚжё…жҘҡеӣҫеғҸе°әеҜёпјҢеҲҷйЎөйқўдјҡеңЁеӣҫеғҸеҠ иҪҪж—¶й—ӘзғҒгҖӮ

е®һдҫӢ

жҠҠеӣҫеғҸе®ҡд№үдёәжҸҗдәӨжҢүй’®пјҢе№¶и®ҫзҪ® height е’Ң width еұһжҖ§пјҡ

е®һдҫӢ

min е’Ң max еұһжҖ§

min е’Ң max еұһжҖ§и§„е®ҡ <input> е…ғзҙ зҡ„жңҖе°ҸеҖје’ҢжңҖеӨ§еҖјгҖӮ

min е’Ң max еұһжҖ§йҖӮз”ЁдәҺеҰӮйңҖиҫ“е…Ҙзұ»еһӢпјҡnumberгҖҒrangeгҖҒdateгҖҒdatetimeгҖҒdatetime-localгҖҒmonthгҖҒtime д»ҘеҸҠ weekгҖӮ

е®һдҫӢ

е…·жңүжңҖе°Ҹе’ҢжңҖеӨ§еҖјзҡ„ <input> е…ғзҙ пјҡ

Enter a date before 1980-01-01:
<input type="date" name="bday" max="1979-12-31">

 Enter a date after 2000-01-01:
<input type="date" name="bday" min="2000-01-02">

 Quantity (between 1 and 5):
<input type="number" name="quantity" min="1" max="5">

еҰӮжһңи®ҫзҪ®пјҢеҲҷи§„е®ҡе…Ғи®ёз”ЁжҲ·еңЁ <input> е…ғзҙ дёӯиҫ“е…ҘдёҖдёӘд»ҘдёҠзҡ„еҖјгҖӮ

multiple еұһжҖ§йҖӮз”ЁдәҺд»ҘдёӢиҫ“е…Ҙзұ»еһӢпјҡemail е’Ң fileгҖӮ

pattern еұһжҖ§

pattern еұһжҖ§и§„е®ҡз”ЁдәҺжЈҖжҹҘ <input> е…ғзҙ еҖјзҡ„жӯЈеҲҷиЎЁиҫҫејҸгҖӮ

pattern еұһжҖ§йҖӮз”ЁдәҺд»ҘдёӢиҫ“е…Ҙзұ»еһӢпјҡtextгҖҒsearchгҖҒurlгҖҒtelгҖҒemailгҖҒand passwordгҖӮ

жҸҗзӨәпјҡиҜ·дҪҝз”Ёе…ЁеұҖзҡ„ title еұһжҖ§еҜ№жЁЎејҸиҝӣиЎҢжҸҸиҝ°д»Ҙеё®еҠ©з”ЁжҲ·гҖӮ

жҸҗзӨәпјҡиҜ·еңЁжҲ‘д»¬зҡ„ JavaScript ж•ҷзЁӢдёӯеӯҰд№ жӣҙеӨҡжңүе…іжӯЈеҲҷиЎЁиҫҫејҸзҡ„зҹҘиҜҶгҖӮ

е®һдҫӢ

еҸӘиғҪеҢ…еҗ«дёүдёӘеӯ—жҜҚзҡ„иҫ“е…Ҙеӯ—ж®өпјҲж— ж•°еӯ—жҲ–зү№ж®Ҡеӯ—з¬Ұпјүпјҡ

placeholder еұһжҖ§

placeholder еұһжҖ§и§„е®ҡз”Ёд»ҘжҸҸиҝ°иҫ“е…Ҙеӯ—ж®өйў„жңҹеҖјзҡ„жҸҗзӨәпјҲж ·жң¬еҖјжҲ–жңүе…іж јејҸзҡ„з®ҖзҹӯжҸҸиҝ°пјүгҖӮ

иҜҘжҸҗзӨәдјҡеңЁз”ЁжҲ·иҫ“е…ҘеҖјд№ӢеүҚжҳҫзӨәеңЁиҫ“е…Ҙеӯ—ж®өдёӯгҖӮ

placeholder еұһжҖ§йҖӮз”ЁдәҺд»ҘдёӢиҫ“е…Ҙзұ»еһӢпјҡtextгҖҒsearchгҖҒurlгҖҒtelгҖҒemail д»ҘеҸҠ passwordгҖӮ

е®һдҫӢ

step еұһжҖ§

зӨәдҫӢпјҡеҰӮжһң step="3"пјҢеҲҷеҗҲжі•ж•°еӯ—еә”иҜҘжҳҜ -3гҖҒ0гҖҒ3гҖҒ6гҖҒзӯүзӯүгҖӮ

жҸҗзӨәпјҡstep еұһжҖ§еҸҜдёҺ max д»ҘеҸҠ min еұһжҖ§дёҖеҗҢдҪҝз”ЁпјҢжқҘеҲӣе»әеҗҲжі•еҖјзҡ„иҢғеӣҙгҖӮ

step еұһжҖ§йҖӮз”ЁдәҺд»ҘдёӢиҫ“е…Ҙзұ»еһӢпјҡnumberгҖҒrangeгҖҒdateгҖҒdatetimeгҖҒdatetime-localгҖҒmonthгҖҒtime д»ҘеҸҠ weekгҖӮ
