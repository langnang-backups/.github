иҫ“е…Ҙзұ»еһӢпјҡnumber

<input type="number"> з”ЁдәҺеә”иҜҘеҢ…еҗ«ж•°еӯ—еҖјзҡ„иҫ“е…Ҙеӯ—ж®өгҖӮ

ж №жҚ®жөҸи§ҲеҷЁж”ҜжҢҒпјҢйҷҗеҲ¶еҸҜеә”з”ЁеҲ°иҫ“е…Ҙеӯ—ж®өгҖӮ

е®һдҫӢ

дәІиҮӘиҜ•дёҖиҜ•

иҫ“е…ҘйҷҗеҲ¶

иҝҷйҮҢеҲ—еҮәдәҶдёҖдәӣеёёз”Ёзҡ„иҫ“е…ҘйҷҗеҲ¶пјҲе…¶дёӯдёҖдәӣжҳҜ HTML5 дёӯж–°еўһзҡ„пјүпјҡ

еұһжҖ§ жҸҸиҝ°
disabled и§„е®ҡиҫ“е…Ҙеӯ—ж®өеә”иҜҘиў«зҰҒз”ЁгҖӮ
max и§„е®ҡиҫ“е…Ҙеӯ—ж®өзҡ„жңҖеӨ§еҖјгҖӮ
maxlength и§„е®ҡиҫ“е…Ҙеӯ—ж®өзҡ„жңҖеӨ§еӯ—з¬Ұж•°гҖӮ
min и§„е®ҡиҫ“е…Ҙеӯ—ж®өзҡ„жңҖе°ҸеҖјгҖӮ
pattern и§„е®ҡйҖҡиҝҮе…¶жЈҖжҹҘиҫ“е…ҘеҖјзҡ„жӯЈеҲҷиЎЁиҫҫејҸгҖӮ
readonly и§„е®ҡиҫ“е…Ҙеӯ—ж®өдёәеҸӘиҜ»пјҲж— жі•дҝ®ж”№пјүгҖӮ
required и§„е®ҡиҫ“е…Ҙеӯ—ж®өжҳҜеҝ…йңҖзҡ„пјҲеҝ…йңҖеЎ«еҶҷпјүгҖӮ
size и§„е®ҡиҫ“е…Ҙеӯ—ж®өзҡ„е®ҪеәҰпјҲд»Ҙеӯ—з¬Ұи®ЎпјүгҖӮ
step и§„е®ҡиҫ“е…Ҙеӯ—ж®өзҡ„еҗҲжі•ж•°еӯ—й—ҙйҡ”гҖӮ
value и§„е®ҡиҫ“е…Ҙеӯ—ж®өзҡ„й»ҳи®ӨеҖјгҖӮ

жӮЁе°ҶеңЁдёӢдёҖз« еӯҰеҲ°жӣҙеӨҡжңүе…іиҫ“е…ҘйҷҗеҲ¶зҡ„зҹҘиҜҶгҖӮ

е®һдҫӢ

<form>
  Quantity:
  <input type="number" name="points" min="0" max="100" step="10" value="30">
</form>
