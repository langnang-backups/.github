д»Җд№ҲжҳҜ HTML5пјҹ

HTML5 жҳҜжңҖж–°зҡ„ HTML ж ҮеҮҶгҖӮ

HTML5 жҳҜдё“й—ЁдёәжүҝиҪҪдё°еҜҢзҡ„ web еҶ…е®№иҖҢи®ҫи®Ўзҡ„пјҢе№¶дё”ж— йңҖйўқеӨ–жҸ’д»¶гҖӮ

HTML5 жӢҘжңүж–°зҡ„иҜӯд№үгҖҒеӣҫеҪўд»ҘеҸҠеӨҡеӘ’дҪ“е…ғзҙ гҖӮ

HTML5 жҸҗдҫӣзҡ„ж–°е…ғзҙ е’Ңж–°зҡ„ API з®ҖеҢ–дәҶ web еә”з”ЁзЁӢеәҸзҡ„жҗӯе»әгҖӮ

HTML5 жҳҜи·Ёе№іеҸ°зҡ„пјҢиў«и®ҫи®ЎдёәеңЁдёҚеҗҢзұ»еһӢзҡ„зЎ¬д»¶пјҲPCгҖҒе№іжқҝгҖҒжүӢжңәгҖҒз”өи§Ҷжңәзӯүзӯүпјүд№ӢдёҠиҝҗиЎҢгҖӮ

жіЁйҮҠпјҡеңЁдёӢйқўзҡ„з« иҠӮдёӯпјҢжӮЁе°ҶеӯҰеҲ°еҰӮдҪ•вҖңеё®еҠ©вҖқиҖҒзүҲжң¬зҡ„жөҸи§ҲеҷЁеӨ„зҗҶ HTML5гҖӮ

HTML5 дёӯзҡ„ж–°еҶ…е®№пјҹ

HTML5 - ж–°зү№жҖ§

HTML5 зҡ„дёҖдәӣжңҖжңүи¶Јзҡ„ж–°зү№жҖ§пјҡ

  • ж–°зҡ„иҜӯд№үе…ғзҙ пјҢжҜ”еҰӮ <header>, <footer>, <article>, and <section>гҖӮ
  • ж–°зҡ„иЎЁеҚ•жҺ§д»¶пјҢжҜ”еҰӮж•°еӯ—гҖҒж—ҘжңҹгҖҒж—¶й—ҙгҖҒж—ҘеҺҶе’Ңж»‘еқ—гҖӮ
  • ејәеӨ§зҡ„еӣҫеғҸж”ҜжҢҒпјҲеҖҹз”ұ <canvas> е’Ң <svg>пјү
  • ејәеӨ§зҡ„еӨҡеӘ’дҪ“ж”ҜжҢҒпјҲеҖҹз”ұ <video> е’Ң <audio>пјү
  • ејәеӨ§зҡ„ж–° APIпјҢжҜ”еҰӮз”Ёжң¬ең°еӯҳеӮЁеҸ–д»Ј cookieгҖӮ
