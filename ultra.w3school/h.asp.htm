HTML

HTML жҢҮи¶…ж–Үжң¬ж ҮзӯҫиҜӯиЁҖгҖӮ

HTML жҳҜйҖҡеҗ‘ WEB жҠҖжңҜдё–з•Ңзҡ„й’ҘеҢҷгҖӮ

еңЁ W3School зҡ„ HTML ж•ҷзЁӢдёӯпјҢжӮЁе°ҶеӯҰд№ еҰӮдҪ•дҪҝз”Ё HTML жқҘеҲӣе»әз«ҷзӮ№гҖӮ

HTML йқһеёёе®№жҳ“еӯҰд№ пјҒдҪ дјҡе–ңж¬ўе®ғзҡ„пјҒзҺ°еңЁејҖе§ӢеӯҰд№  HTML пјҒ

CSS

CSS жҢҮеұӮеҸ ж ·ејҸиЎЁпјҲCascading Style SheetsпјүгҖӮ

еңЁжҲ‘д»¬зҡ„ CSS ж•ҷзЁӢдёӯпјҢжӮЁе°ҶеӯҰд№ еҰӮдҪ•дҪҝз”Ё CSS еҗҢж—¶жҺ§еҲ¶ж•ҙдёӘз«ҷзӮ№зҡ„ж ·ејҸе’ҢеёғеұҖгҖӮ

йҖҡиҝҮдёҺ XHTML з»“еҗҲпјҢCSS еҸҜд»Ҙеё®еҠ©жҲ‘д»¬е®һиЎҢиЎЁзҺ°дёҺз»“жһ„еҲҶзҰ»зҡ„ејҖеҸ‘жЁЎејҸгҖӮ

йҖҡиҝҮдҪҝз”Ё CSS жқҘжҸҗеҚҮе·ҘдҪңж•ҲзҺҮпјҒзҺ°еңЁејҖе§ӢеӯҰд№  CSS пјҒ

Bootstrap 5

Bootstrap жҳҜжңҖжөҒиЎҢзҡ„ HTMLгҖҒCSS е’Ң JavaScript жЎҶжһ¶пјҢз”ЁдәҺеҲӣе»әе“Қеә”ејҸгҖҒз§»еҠЁдјҳе…Ҳзҡ„зҪ‘з«ҷгҖӮ

Bootstrap 5 жҳҜ Bootstrap зҡ„жңҖж–°зүҲжң¬гҖӮ

жҲ‘д»¬зҡ„ Bootstrap 5 ж•ҷзЁӢжҸҗдҫӣж•°зҷҫдёӘ Bootstrap 5 е®һдҫӢгҖӮ

зҺ°еңЁе°ұејҖе§ӢеӯҰд№  Bootstrap 5 пјҒ

W3School зҷҫе®қз®ұ

еҺҹз”ҹ HTMLгҖҒCSSгҖҒJavaScript жү“йҖ зҡ„д»Јз Ғеә“гҖӮ

ж¶өзӣ–иҸңеҚ•гҖҒеӣҫеғҸгҖҒжҢүй’®гҖҒиЎЁеҚ•гҖҒиЎЁж јгҖҒзҪ‘ж јеёғеұҖзӯүдәӨдә’е…ғзҙ е’Ңж ·ејҸгҖӮ

жҸҗдҫӣж•°зҷҫдёӘжЎҲдҫӢпјҢж•°еҚғд»Јз ҒзүҮж®өгҖӮ

еҲҶжӯҘйӘӨи®Іи§ЈпјҢжіЁйҮҠйқһеёёжё…жҷ°пјҢи®ҫи®ЎзҫҺи§ӮпјҢеҸҜз”ЁжҖ§еҫҲй«ҳпјҒжү“ејҖзҷҫе®қз®ұ пјҒ

XHTML

XHTML жҳҜжӣҙдёҘи°ЁжӣҙзәҜеҮҖзҡ„ HTML зүҲжң¬гҖӮ

еңЁ W3School зҡ„ XHTML ж•ҷзЁӢдёӯпјҢжӮЁе°ҶеӯҰд№  HTML дёҺ XHTML д№Ӣй—ҙзҡ„е·®ејӮпјҢд»ҘеҸҠ XHTML зҡ„дјҳеҠҝжүҖеңЁгҖӮ

жҲ‘д»¬иҝҳе°ҶдјҡдёәжӮЁеұ•зӨәе°Ҷз«ҷзӮ№еҚҮзә§иҮі XHTML зҡ„ж–№жі•пјҢд»Ҙеё®еҠ©жӮЁеҝ«йҖҹйғЁзҪІ XHTML жҠҖжңҜгҖӮ

XHTML е·Із»ҸжҲҗдёәдјҳз§ҖеүҚз«Ҝи®ҫи®ЎеёҲе’Ңе·ҘзЁӢеёҲзҡ„йҰ–йҖүгҖӮзҺ°еңЁејҖе§ӢеӯҰд№  XHTML пјҒ

HTML 5

HTML 5 жҳҜдёӢдёҖд»Јзҡ„ HTMLгҖӮ

HTML5 д»ҚеӨ„дәҺе®Ңе–„д№ӢдёӯгҖӮз„¶иҖҢпјҢеӨ§йғЁеҲҶзҺ°д»ЈжөҸи§ҲеҷЁе·Із»Ҹе…·еӨҮдәҶжҹҗдәӣ HTML5 ж”ҜжҢҒгҖӮ

еңЁ W3School зҡ„ HTML 5 ж•ҷзЁӢдёӯпјҢжӮЁе°ҶдәҶи§Ј HTML5 дёӯзҡ„ж–°зү№жҖ§гҖӮ

зҺ°еңЁе°ұејҖе§ӢеӯҰд№  HTML 5 пјҒ

CSS3

CSS3 жҳҜжңҖж–°зҡ„ CSS ж ҮеҮҶгҖӮ

жҲ‘д»¬зҡ„ CSS3 ж•ҷзЁӢеҗ‘жӮЁи®Іи§Ј CSS3 дёӯзҡ„ж–°зү№жҖ§гҖӮ

зҺ°еңЁе°ұејҖе§ӢеӯҰд№  CSS3 пјҒ

TCP/IP

TCP/IP жҳҜй’ҲеҜ№еӣ зү№зҪ‘зҡ„йҖҡдҝЎеҚҸи®®гҖӮ

еңЁ W3School зҡ„ TCP/IP ж•ҷзЁӢдёӯпјҢдҪ е°ҶдәҶи§ЈеҲ°д»Җд№ҲжҳҜ TCP/IPпјҢд»ҘеҸҠе®ғеҰӮдҪ•е·ҘдҪңгҖӮ

е…ідәҺ TCP/IP зҹҘиҜҶпјҢжңүеҠ©дәҺжӮЁзҗҶи§Јдә’иҒ”зҪ‘зҡ„иҝҗиЎҢеҺҹзҗҶпјҢжҳҜжӮЁеӯҰд№  WEB жҠҖжңҜзҡ„еҹәзЎҖгҖӮ

зҺ°еңЁејҖе§ӢеӯҰд№  TCP/IP пјҒ
