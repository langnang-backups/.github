Sass еҸҜдҝқжҢҒ CSS д»Јз Ғе№ІзҮҘпјҲDRYпјҢDon't Repeat YourselfпјҢдёҚиҰҒйҮҚеӨҚиҮӘе·ұпјүгҖӮ

зј–еҶҷ DRY д»Јз Ғзҡ„дёҖз§Қж–№жі•жҳҜе°Ҷзӣёе…ід»Јз ҒдҝқеӯҳеңЁеҚ•зӢ¬зҡ„ж–Үд»¶дёӯгҖӮ

жӮЁеҸҜдҪҝз”Ё CSS зүҮж®өеҲӣе»әе°Ҹж–Үд»¶пјҢе№¶еҢ…еҗ«еңЁе…¶д»– Sass ж–Үд»¶дёӯгҖӮдҫӢеҰӮиҝҷзұ»ж–Үд»¶еҸҜд»ҘжҳҜпјҡйҮҚзҪ®ж–Үд»¶гҖҒеҸҳйҮҸгҖҒйўңиүІгҖҒеӯ—дҪ“зӯүгҖӮ

Sass ж–Үд»¶еҜје…Ҙ

е°ұеғҸ CSS дёҖж ·пјҢSass д№ҹж”ҜжҢҒ @import жҢҮд»ӨгҖӮ

@import жҢҮд»Өе…Ғи®ёжӮЁе°ҶдёҖдёӘж–Үд»¶зҡ„еҶ…е®№еҢ…еҗ«еңЁеҸҰдёҖдёӘж–Үд»¶дёӯгҖӮ

з”ұдәҺжҖ§иғҪй—®йўҳпјҢCSS @import жҢҮд»ӨжңүдёҖдёӘдё»иҰҒзҡ„зјәзӮ№пјӣжҜҸж¬Ўи°ғз”Ёе®ғж—¶йғҪдјҡеҲӣе»әдёҖдёӘйўқеӨ–зҡ„ HTTP иҜ·жұӮгҖӮдҪҶжҳҜпјҢSass @import жҢҮд»Өе°Ҷж–Үд»¶еҢ…еҗ«еңЁ CSS дёӯпјӣжүҖд»ҘеңЁиҝҗиЎҢж—¶дёҚйңҖиҰҒйўқеӨ–зҡ„ HTTP и°ғз”ЁпјҒ

Sass еҜје…ҘиҜӯжі•пјҡ

@import filename;

жҸҗзӨәпјҡжӮЁдёҚйңҖиҰҒжҢҮе®ҡж–Үд»¶жү©еұ•еҗҚпјҢSass дјҡиҮӘеҠЁеҒҮе®ҡжӮЁжҢҮзҡ„жҳҜ .sass жҲ– .scss ж–Үд»¶гҖӮжӮЁиҝҳеҸҜд»ҘеҜје…Ҙ CSS ж–Үд»¶гҖӮ@import жҢҮд»ӨеҜје…Ҙж–Үд»¶пјҢз„¶еҗҺеҸҜеңЁдё»ж–Үд»¶дёӯдҪҝз”ЁеҜје…Ҙж–Үд»¶дёӯе®ҡд№үзҡ„д»»дҪ•еҸҳйҮҸжҲ– mixinгҖӮ

жӮЁеҸҜд»ҘеңЁдё»ж–Үд»¶дёӯеҜје…Ҙд»»ж„Ҹж•°йҮҸзҡ„ж–Үд»¶пјҡ

е®һдҫӢ

@import "variables";
@import "colors";
@import "reset";

и®©жҲ‘д»¬зңӢдёҖдёӘдҫӢеӯҗпјҡеҒҮи®ҫжҲ‘д»¬жңүдёҖдёӘеҗҚдёә "reset.scss" зҡ„йҮҚзҪ®ж–Үд»¶пјҢе®ғзңӢиө·жқҘжҳҜиҝҷж ·зҡ„пјҡ

SCSS иҜӯжі• (reset.scss)пјҡ

html,
body,
ul,
ol {
  margin: 0;
  padding: 0;
}

зҺ°еңЁжҲ‘д»¬иҰҒе°Ҷ "reset.scss" ж–Үд»¶еҜје…ҘеҲ°еҸҰдёҖдёӘеҗҚдёә "standard.scss" зҡ„ж–Үд»¶дёӯгҖӮ

жҲ‘д»¬жҳҜиҝҷж ·еҒҡзҡ„пјҡдёҖиҲ¬жҳҜеңЁж–Үд»¶йЎ¶йғЁж·»еҠ  @import жҢҮд»Өпјӣиҝҷж ·е…¶еҶ…е®№е°Ҷе…·жңүе…ЁеұҖдҪңз”Ёеҹҹпјҡ

SCSS иҜӯжі• (standard.scss):

@import "reset";

body {
  font-family: Helvetica, sans-serif;
  font-size: 18px;
  color: red;
}

еӣ жӯӨпјҢеҪ“ "standard.css" ж–Үд»¶иў«иҪ¬иҜ‘ж—¶пјҢCSS е°ҶеҰӮдёӢжүҖзӨәпјҡ

CSS иҫ“еҮәпјҡ

html, body, ul, ol {
  margin: 0;
  padding: 0;
}

body {
  font-family: Helvetica, sans-serif;
  font-size: 18px;
  color: red;
}

Sass PartialsпјҲеұҖйғЁж–Үд»¶пјү

й»ҳи®Өжғ…еҶөдёӢпјҢSass зӣҙжҺҘиҪ¬иҜ‘жүҖжңү .scss ж–Үд»¶гҖӮдҪҶжҳҜпјҢеңЁеҜје…Ҙж–Үд»¶ж—¶пјҢжӮЁдёҚйңҖиҰҒж–Үд»¶иў«зӣҙжҺҘиҪ¬иҜ‘гҖӮ

Sass жңүдёҖдёӘжңәеҲ¶пјҡеҰӮжһңдҪ з”ЁдёӢеҲ’зәҝејҖе§Ӣж–Үд»¶еҗҚпјҢSass е°ҶдёҚдјҡиҪ¬иҜ‘е®ғгҖӮд»Ҙиҝҷз§Қж–№ејҸе‘ҪеҗҚзҡ„ж–Үд»¶еңЁ Sass дёӯз§°дёә partialsгҖӮ

еӣ жӯӨпјҢpartial Sass ж–Үд»¶дҪҝз”ЁеүҚеҜјдёӢеҲ’зәҝе‘ҪеҗҚпјҡ

Sass Partial иҜӯжі•пјҡ

_filename;

дёӢдҫӢжҳҫзӨәдәҶеҗҚдёә "_colors.scss" зҡ„ partial Sass ж–Үд»¶гҖӮпјҲжӯӨж–Үд»¶дёҚдјҡзӣҙжҺҘиҪ¬жҚўдёә "colors.css"пјүпјҡ

е®һдҫӢ

"_colors.scss":

$myPink: #EE82EE;
$myBlue: #4169E1;
$myGreen: #8FBC8F;

зҺ°еңЁпјҢеҰӮжһңжӮЁеҜје…ҘиҜҘ partial ж–Үд»¶пјҢиҜ·зңҒз•ҘдёӢеҲ’зәҝгҖӮSass зҡ„зҗҶи§ЈжҳҜе®ғеә”иҜҘеҜје…Ҙж–Үд»¶ "_colors.scss"пјҡ
