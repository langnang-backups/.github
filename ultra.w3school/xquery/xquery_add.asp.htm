XML е®һдҫӢж–ҮжЎЈ

жҲ‘д»¬е°ҶеңЁдёӢйқўзҡ„дҫӢеӯҗдёӯз»§з»ӯдҪҝз”ЁиҝҷдёӘ "books.xml" ж–ҮжЎЈпјҲе’ҢдёҠйқўзҡ„з« иҠӮжүҖдҪҝз”Ёзҡ„ XML ж–Үд»¶зӣёеҗҢпјүгҖӮ

еңЁжӮЁзҡ„жөҸи§ҲеҷЁдёӯжҹҘзңӢ "books.xml" ж–Үд»¶гҖӮ

еҗ‘з»“жһңж·»еҠ е…ғзҙ е’ҢеұһжҖ§

жӯЈеҰӮеңЁеүҚйқўдёҖиҠӮзңӢеҲ°зҡ„пјҢжҲ‘д»¬еҸҜд»ҘеңЁз»“жһңдёӯеј•з”Ёиҫ“е…Ҙж–Үд»¶дёӯзҡ„е…ғзҙ е’ҢеұһжҖ§пјҡ

for $x in doc("books.xml")/bookstore/book/title
order by $x
return $x

дёҠйқўзҡ„ XQuery иЎЁиҫҫејҸдјҡеңЁз»“жһңдёӯеј•з”Ё title е…ғзҙ е’Ң lang еұһжҖ§пјҢе°ұеғҸиҝҷж ·пјҡ

<title lang="en">Everyday Italian</title>
<title lang="en">Harry Potter</title>
<title lang="en">Learning XML</title>
<title lang="en">XQuery Kick Start</title>

д»ҘдёҠ XQuery иЎЁиҫҫејҸиҝ”еӣһ title е…ғзҙ зҡ„ж–№ејҸе’Ңе®ғд»¬еңЁиҫ“е…Ҙж–ҮжЎЈдёӯиў«жҸҸиҝ°зҡ„ж–№ејҸзҡ„зӣёеҗҢзҡ„гҖӮ

зҺ°еңЁжҲ‘д»¬иҰҒеҗ‘з»“жһңж·»еҠ жҲ‘д»¬иҮӘе·ұзҡ„е…ғзҙ е’ҢеұһжҖ§пјҒ

ж·»еҠ  HTML е…ғзҙ е’Ңж–Үжң¬

зҺ°еңЁпјҢжҲ‘д»¬иҰҒеҗ‘з»“жһңж·»еҠ  HTML е…ғзҙ гҖӮжҲ‘д»¬дјҡжҠҠз»“жһңж”ҫеңЁдёҖдёӘ HTML еҲ—иЎЁдёӯпјҡ

<html>
<body>

<h1>Bookstore</h1>

<ul>
{
for $x in doc("books.xml")/bookstore/book
order by $x/title
return <li>{data($x/title)}. Category: {data($x/@category)}</li>
}
</ul>

</body>
</html>

д»ҘдёҠ XQuery иЎЁиҫҫејҸдјҡз”ҹжҲҗдёӢйқўзҡ„з»“жһңпјҡ

<html>
<body>

<h1>Bookstore</h1>

<ul>
<li>Everyday Italian. Category: COOKING</li>
<li>Harry Potter. Category: CHILDREN</li>
<li>Learning XML. Category: WEB</li>
<li>XQuery Kick Start. Category: WEB</li>
</ul>

</body>
</html>

еҗ‘ HTML е…ғзҙ ж·»еҠ еұһжҖ§

жҺҘдёӢжқҘпјҢжҲ‘д»¬иҰҒжҠҠ category еұһжҖ§дҪңдёә HTML еҲ—иЎЁдёӯзҡ„ class еұһжҖ§жқҘдҪҝз”Ёпјҡ

<html>
<body>

<h1>Bookstore</h1>

<ul>
{
for $x in doc("books.xml")/bookstore/book
order by $x/title
return <li class="{data($x/@category)}">{data($x/title)}</li>
}
</ul>

</body>
</html>

дёҠйқўзҡ„ XQuery иЎЁиҫҫејҸеҸҜз”ҹжҲҗд»ҘдёӢз»“жһңпјҡ

<html>
<body>

<h1>Bookstore</h1>

<ul>
<li class="COOKING">Everyday Italian</li>
<li class="CHILDREN">Harry Potter</li>
<li class="WEB">Learning XML</li>
<li class="WEB">XQuery Kick Start</li>
</ul>

</body>
</html>
