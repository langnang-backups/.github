XPath дҪҝз”Ёи·Ҝеҫ„иЎЁиҫҫејҸжқҘйҖүеҸ– XML ж–ҮжЎЈдёӯзҡ„иҠӮзӮ№жҲ–иҠӮзӮ№йӣҶгҖӮиҠӮзӮ№жҳҜйҖҡиҝҮжІҝзқҖи·Ҝеҫ„ (path) жҲ–иҖ…жӯҘ (steps) жқҘйҖүеҸ–зҡ„гҖӮ

XML е®һдҫӢж–ҮжЎЈ

жҲ‘д»¬е°ҶеңЁдёӢйқўзҡ„дҫӢеӯҗдёӯдҪҝз”ЁиҝҷдёӘ XML ж–ҮжЎЈгҖӮ

<?xml version="1.0" encoding="ISO-8859-1"?>

<bookstore>

  <title lang="eng">Harry Potter</title>
  <price>29.99</price>
</book>

<book>
  <title lang="eng">Learning XML</title>
  <price>39.95</price>
</book>

</bookstore>

йҖүеҸ–иҠӮзӮ№

XPath дҪҝз”Ёи·Ҝеҫ„иЎЁиҫҫејҸеңЁ XML ж–ҮжЎЈдёӯйҖүеҸ–иҠӮзӮ№гҖӮиҠӮзӮ№жҳҜйҖҡиҝҮжІҝзқҖи·Ҝеҫ„жҲ–иҖ… step жқҘйҖүеҸ–зҡ„гҖӮ

дёӢйқўеҲ—еҮәдәҶжңҖжңүз”Ёзҡ„и·Ҝеҫ„иЎЁиҫҫејҸпјҡ

иЎЁиҫҫејҸ жҸҸиҝ°
nodename йҖүеҸ–жӯӨиҠӮзӮ№зҡ„жүҖжңүеӯҗиҠӮзӮ№гҖӮ
/ д»Һж №иҠӮзӮ№йҖүеҸ–гҖӮ
// д»ҺеҢ№й…ҚйҖүжӢ©зҡ„еҪ“еүҚиҠӮзӮ№йҖүжӢ©ж–ҮжЎЈдёӯзҡ„иҠӮзӮ№пјҢиҖҢдёҚиҖғиҷ‘е®ғд»¬зҡ„дҪҚзҪ®гҖӮ
. йҖүеҸ–еҪ“еүҚиҠӮзӮ№гҖӮ
.. йҖүеҸ–еҪ“еүҚиҠӮзӮ№зҡ„зҲ¶иҠӮзӮ№гҖӮ
@ йҖүеҸ–еұһжҖ§гҖӮ

е®һдҫӢ

еңЁдёӢйқўзҡ„иЎЁж јдёӯпјҢжҲ‘д»¬е·ІеҲ—еҮәдәҶдёҖдәӣи·Ҝеҫ„иЎЁиҫҫејҸд»ҘеҸҠиЎЁиҫҫејҸзҡ„з»“жһңпјҡ

и·Ҝеҫ„иЎЁиҫҫејҸ з»“жһң

/bookstore

йҖүеҸ–ж №е…ғзҙ  bookstoreгҖӮ

жіЁйҮҠпјҡеҒҮеҰӮи·Ҝеҫ„иө·е§ӢдәҺжӯЈж–ңжқ ( / )пјҢеҲҷжӯӨи·Ҝеҫ„е§Ӣз»Ҳд»ЈиЎЁеҲ°жҹҗе…ғзҙ зҡ„з»қеҜ№и·Ҝеҫ„пјҒ
bookstore/book йҖүеҸ–еұһдәҺ bookstore зҡ„еӯҗе…ғзҙ зҡ„жүҖжңү book е…ғзҙ гҖӮ
//book йҖүеҸ–жүҖжңү book еӯҗе…ғзҙ пјҢиҖҢдёҚз®Ўе®ғд»¬еңЁж–ҮжЎЈдёӯзҡ„дҪҚзҪ®гҖӮ
bookstore//book йҖүжӢ©еұһдәҺ bookstore е…ғзҙ зҡ„еҗҺд»Јзҡ„жүҖжңү book е…ғзҙ пјҢиҖҢдёҚз®Ўе®ғд»¬дҪҚдәҺ bookstore д№ӢдёӢзҡ„д»Җд№ҲдҪҚзҪ®гҖӮ
//@lang йҖүеҸ–еҗҚдёә lang зҡ„жүҖжңүеұһжҖ§гҖӮ

и°“иҜӯпјҲPredicatesпјү

и°“иҜӯз”ЁжқҘжҹҘжүҫжҹҗдёӘзү№е®ҡзҡ„иҠӮзӮ№жҲ–иҖ…еҢ…еҗ«жҹҗдёӘжҢҮе®ҡзҡ„еҖјзҡ„иҠӮзӮ№гҖӮ

и°“иҜӯиў«еөҢеңЁж–№жӢ¬еҸ·дёӯгҖӮ

е®һдҫӢ

еңЁдёӢйқўзҡ„иЎЁж јдёӯпјҢжҲ‘д»¬еҲ—еҮәдәҶеёҰжңүи°“иҜӯзҡ„дёҖдәӣи·Ҝеҫ„иЎЁиҫҫејҸпјҢд»ҘеҸҠиЎЁиҫҫејҸзҡ„з»“жһңпјҡ

и·Ҝеҫ„иЎЁиҫҫејҸ з»“жһң
/bookstore/book[1] йҖүеҸ–еұһдәҺ bookstore еӯҗе…ғзҙ зҡ„з¬¬дёҖдёӘ book е…ғзҙ гҖӮ
/bookstore/book[last()] йҖүеҸ–еұһдәҺ bookstore еӯҗе…ғзҙ зҡ„жңҖеҗҺдёҖдёӘ book е…ғзҙ гҖӮ
/bookstore/book[last()-1] йҖүеҸ–еұһдәҺ bookstore еӯҗе…ғзҙ зҡ„еҖ’ж•°з¬¬дәҢдёӘ book е…ғзҙ гҖӮ
/bookstore/book[position()<3] йҖүеҸ–жңҖеүҚйқўзҡ„дёӨдёӘеұһдәҺ bookstore е…ғзҙ зҡ„еӯҗе…ғзҙ зҡ„ book е…ғзҙ гҖӮ
//title[@lang] йҖүеҸ–жүҖжңүжӢҘжңүеҗҚдёә lang зҡ„еұһжҖ§зҡ„ title е…ғзҙ гҖӮ
//title[@lang='eng'] йҖүеҸ–жүҖжңү title е…ғзҙ пјҢдё”иҝҷдәӣе…ғзҙ жӢҘжңүеҖјдёә eng зҡ„ lang еұһжҖ§гҖӮ
/bookstore/book[price>35.00] йҖүеҸ– bookstore е…ғзҙ зҡ„жүҖжңү book е…ғзҙ пјҢдё”е…¶дёӯзҡ„ price е…ғзҙ зҡ„еҖјйЎ»еӨ§дәҺ 35.00гҖӮ
/bookstore/book[price>35.00]/title йҖүеҸ– bookstore е…ғзҙ дёӯзҡ„ book е…ғзҙ зҡ„жүҖжңү title е…ғзҙ пјҢдё”е…¶дёӯзҡ„ price е…ғзҙ зҡ„еҖјйЎ»еӨ§дәҺ 35.00гҖӮ

йҖүеҸ–жңӘзҹҘиҠӮзӮ№

XPath йҖҡй…Қз¬ҰеҸҜз”ЁжқҘйҖүеҸ–жңӘзҹҘзҡ„ XML е…ғзҙ гҖӮ

йҖҡй…Қз¬Ұ жҸҸиҝ°

е®һдҫӢ

еңЁдёӢйқўзҡ„иЎЁж јдёӯпјҢжҲ‘д»¬еҲ—еҮәдәҶдёҖдәӣи·Ҝеҫ„иЎЁиҫҫејҸпјҢд»ҘеҸҠиҝҷдәӣиЎЁиҫҫејҸзҡ„з»“жһңпјҡ
