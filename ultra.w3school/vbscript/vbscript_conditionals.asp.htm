If....Then.....Else

еңЁдёӢйқўзҡ„жғ…еҶөдёӯпјҢжӮЁеҸҜд»ҘдҪҝз”Ё If...Then...Else иҜӯеҸҘпјҡ

  • еңЁжқЎд»¶дёә true ж—¶пјҢжү§иЎҢжҹҗж®өд»Јз Ғ
  • йҖүжӢ©дёӨж®өд»Јз Ғд№ӢдёҖжқҘжү§иЎҢж—¶

еҰӮжһңйңҖиҰҒеңЁжқЎд»¶дёә true ж—¶еҸӘжү§иЎҢдёҖиЎҢиҜӯеҸҘпјҢеҸҜд»ҘжҠҠд»Јз ҒеҶҷдёәдёҖиЎҢпјҡ

if i=10 Then msgbox "Hello"

еңЁдёҠйқўзҡ„д»Јз ҒдёӯпјҢжІЎжңү .else.. иҜӯеҸҘгҖӮжҲ‘д»¬д»…д»…и®©д»Јз ҒеңЁжқЎд»¶дёә true ж—¶жү§иЎҢдёҖйЎ№ж“ҚдҪңпјҲеҪ“ i дёә 10 ж—¶пјүгҖӮ

еҒҮеҰӮжҲ‘д»¬йңҖиҰҒеңЁжқЎд»¶дёә true ж—¶жү§иЎҢдёҚжӯўдёҖжқЎиҜӯеҸҘпјҢйӮЈд№Ҳе°ұеҝ…йЎ»еңЁдёҖиЎҢеҶҷдёҖжқЎиҜӯеҸҘпјҢз„¶еҗҺдҪҝз”Ёе…ій”®иҜҚ "End If" жқҘз»“жқҹиҝҷдёӘиҜӯеҸҘпјҡ

if i=10 Then
   msgbox "Hello"
   i = i+1
end If

еңЁдёҠйқўзҡ„д»Јз ҒдёӯпјҢеҗҢж ·жІЎжңү .else.. иҜӯеҸҘгҖӮжҲ‘д»¬д»…д»…и®©д»Јз ҒеңЁжқЎд»¶дёә true ж—¶жү§иЎҢдәҶеӨҡйЎ№ж“ҚдҪңгҖӮ

еҒҮеҰӮжҲ‘д»¬еёҢжңӣеңЁжқЎд»¶дёә true ж—¶жү§иЎҢжҹҗжқЎиҜӯеҸҘпјҢе№¶еҪ“жқЎд»¶дёҚдёә true ж—¶жү§иЎҢеҸҰдёҖжқЎиҜӯеҸҘпјҢе°ұеҝ…йЎ»ж·»еҠ е…ій”®иҜҚ "Else"пјҡ

if i=10 then
   msgbox "Hello"
else
   msgbox "Goodbye"
end If

еҪ“жқЎд»¶дёә true ж—¶дјҡжү§иЎҢз¬¬дёҖж®өд»Јз ҒпјҢеҪ“жқЎд»¶дёҚжҲҗз«Ӣж—¶жү§иЎҢз¬¬дәҢж®өд»Јз ҒпјҲеҪ“ i дёҚзӯүдәҺ 10 ж—¶пјүгҖӮ

If....Then.....Elseif

еҒҮеҰӮдҪ еёҢжңӣйҖүжӢ©еӨҡеҘ—д»Јз Ғд№ӢдёҖжқҘжү§иЎҢпјҢеҸҜд»ҘдҪҝз”Ёif...then...elseifиҜӯеҸҘпјҡ

if payment="Cash" then
   msgbox "You are going to pay cash!"
 elseif payment="Visa" then
   msgbox "You are going to pay with visa."
 elseif payment="AmEx" then
   msgbox "You are going to pay with American Express."
 else
   msgbox "Unknown method of payment."
end If

Select Case

еҒҮеҰӮдҪ еёҢжңӣйҖүжӢ©еӨҡеҘ—д»Јз Ғд№ӢдёҖжқҘжү§иЎҢпјҢеҸҜд»ҘдҪҝз”Ё SELECT иҜӯеҸҘпјҡ

select case payment
 case "Cash"
   msgbox "You are going to pay cash"
 case "Visa"
   msgbox "You are going to pay with visa"
 case "AmEx"
   msgbox "You are going to pay with American Express"
 case Else
   msgbox "Unknown method of payment"
end select
