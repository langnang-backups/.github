зҪ‘ж је®№еҷЁз”ұж”ҫзҪ®еңЁеҲ—е’ҢиЎҢеҶ…зҡ„зҪ‘ж јйЎ№зӣ®з»„жҲҗгҖӮ

grid-template-columns еұһжҖ§

grid-template-columns еұһжҖ§е®ҡд№үзҪ‘ж јеёғеұҖдёӯзҡ„еҲ—ж•°пјҢе№¶еҸҜе®ҡд№үжҜҸеҲ—зҡ„е®ҪеәҰгҖӮ

иҜҘеҖјжҳҜд»Ҙз©әж јеҲҶйҡ”зҡ„еҲ—иЎЁпјҢе…¶дёӯжҜҸдёӘеҖје®ҡд№үзӣёеә”еҲ—зҡ„й•ҝеәҰгҖӮ

еҰӮжһңжӮЁеёҢжңӣзҪ‘ж јеёғеұҖеҢ…еҗ« 4 еҲ—пјҢиҜ·жҢҮе®ҡиҝҷ 4 еҲ—зҡ„е®ҪеәҰпјӣеҰӮжһңжүҖжңүеҲ—йғҪеә”еҪ“жңүзӣёеҗҢзҡ„е®ҪеәҰпјҢеҲҷи®ҫзҪ®дёә "auto"гҖӮ

е®һдҫӢ

з”ҹжҲҗеҢ…еҗ«еӣӣеҲ—зҡ„зҪ‘ж јпјҡ

.grid-container {
  display: grid;
  grid-template-columns: auto auto auto auto;
}

жіЁж„ҸпјҡеҰӮжһңеңЁ 4 еҲ—зҪ‘ж јдёӯжңү 4 дёӘд»ҘдёҠзҡ„йЎ№зӣ®пјҢеҲҷзҪ‘ж јдјҡиҮӘеҠЁж·»еҠ ж–°иЎҢе№¶е°ҶиҝҷдәӣйЎ№зӣ®ж”ҫе…Ҙе…¶дёӯгҖӮ

grid-template-columns еұһжҖ§иҝҳеҸҜд»Ҙз”ЁдәҺжҢҮе®ҡеҲ—зҡ„е°әеҜёпјҲе®ҪеәҰпјүгҖӮ
