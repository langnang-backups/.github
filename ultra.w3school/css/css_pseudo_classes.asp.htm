д»Җд№ҲжҳҜдјӘзұ»пјҹ

дјӘзұ»з”ЁдәҺе®ҡд№үе…ғзҙ зҡ„зү№ж®ҠзҠ¶жҖҒгҖӮ

дҫӢеҰӮпјҢе®ғеҸҜд»Ҙз”ЁдәҺпјҡ

  • и®ҫзҪ®йј ж ҮжӮ¬еҒңеңЁе…ғзҙ дёҠж—¶зҡ„ж ·ејҸ
  • дёәе·Іи®ҝй—®е’ҢжңӘи®ҝй—®й“ҫжҺҘи®ҫзҪ®дёҚеҗҢзҡ„ж ·ејҸ
  • и®ҫзҪ®е…ғзҙ иҺ·еҫ—з„ҰзӮ№ж—¶зҡ„ж ·ејҸ
иҜ·е°Ҷйј ж ҮжӮ¬еҒңеңЁжҲ‘дёҠйқў

иҜӯжі•

дјӘзұ»зҡ„иҜӯжі•пјҡ

selector:pseudo-class {
  property: value;
}

й”ҡдјӘзұ»

й“ҫжҺҘиғҪеӨҹд»ҘдёҚеҗҢзҡ„ж–№ејҸжҳҫзӨәпјҡ

е®һдҫӢ

/* жңӘи®ҝй—®зҡ„й“ҫжҺҘ */
a:link {
  color: #FF0000;
}

/* е·Іи®ҝй—®зҡ„й“ҫжҺҘ */
a:visited {
  color: #00FF00;
}

/* йј ж ҮжӮ¬еҒңй“ҫжҺҘ */
a:hover {
  color: #FF00FF;
}

/* е·ІйҖүжӢ©зҡ„й“ҫжҺҘ */
a:active {
  color: #0000FF;
}

дәІиҮӘиҜ•дёҖиҜ•

жіЁж„Ҹпјҡa:hover еҝ…йЎ»еңЁ CSS е®ҡд№үдёӯзҡ„ a:link е’Ң a:visited д№ӢеҗҺпјҢжүҚиғҪз”ҹж•ҲпјҒa:active еҝ…йЎ»еңЁ CSS е®ҡд№үдёӯзҡ„ a:hover д№ӢеҗҺжүҚиғҪз”ҹж•ҲпјҒдјӘзұ»еҗҚз§°еҜ№еӨ§е°ҸеҶҷдёҚж•Ҹж„ҹгҖӮ

дјӘзұ»е’Ң CSS зұ»

дјӘзұ»еҸҜд»ҘдёҺ CSS зұ»з»“еҗҲдҪҝз”Ёпјҡ

еҪ“жӮЁе°Ҷйј ж ҮжӮ¬еҒңеңЁдҫӢеӯҗдёӯзҡ„й“ҫжҺҘдёҠж—¶пјҢе®ғдјҡж”№еҸҳйўңиүІпјҡ

е®һдҫӢ

a.highlight:hover {
  color: #ff0000;
}

дәІиҮӘиҜ•дёҖиҜ•

жӮ¬еҒңеңЁ <div> дёҠ

еңЁ <div> е…ғзҙ дёҠдҪҝз”Ё :hover дјӘзұ»зҡ„е®һдҫӢпјҡ

е®һдҫӢ

div:hover {
  background-color: blue;
}

дәІиҮӘиҜ•дёҖиҜ•

з®ҖеҚ•зҡ„е·Ҙе…·жҸҗзӨәжӮ¬еҒң

жҠҠйј ж ҮжӮ¬еҒңеҲ° <div> е…ғзҙ д»ҘжҳҫзӨә <p> е…ғзҙ пјҲзұ»дјје·Ҙе…·жҸҗзӨәзҡ„ж•Ҳжһңпјүпјҡ

жӮ¬еҒңеҲ°жҲ‘дёҠйқўжқҘжҳҫзӨә <p> е…ғзҙ гҖӮ

еҳҝеҳҝпјҢжҲ‘еңЁиҝҷйҮҢпјҒ

е®һдҫӢ

p {
  display: none;
  background-color: yellow;
  padding: 20px;
}

div:hover p {
  display: block;
}

еҢ№й…ҚйҰ–дёӘ <p> е…ғзҙ

  color: blue;

дәІиҮӘиҜ•дёҖиҜ•

еҢ№й…ҚжүҖжңүйҰ–дёӘ <p> е…ғзҙ дёӯзҡ„жүҖжңү <i> е…ғзҙ 

еңЁдёӢйқўзҡ„дҫӢеӯҗдёӯпјҢйҖүжӢ©еҷЁеҢ№й…ҚдҪңдёәеҸҰдёҖдёӘе…ғзҙ зҡ„з¬¬дёҖдёӘеӯҗе…ғзҙ зҡ„ <p> е…ғзҙ дёӯзҡ„жүҖжңү <i> е…ғзҙ пјҡ

е®һдҫӢ

p:first-child i {
  color: blue;

CSS - :lang дјӘзұ»

:lang дјӘзұ»е…Ғи®ёжӮЁдёәдёҚеҗҢзҡ„иҜӯиЁҖе®ҡд№үзү№ж®Ҡзҡ„и§„еҲҷгҖӮ

еңЁдёӢйқўзҡ„дҫӢеӯҗдёӯпјҢ:lang дёәеұһжҖ§дёә lang="en" зҡ„ <q> е…ғзҙ е®ҡд№үеј•еҸ·пјҡ
