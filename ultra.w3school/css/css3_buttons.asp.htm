еӯҰд№ еҰӮдҪ•дҪҝз”Ё CSS и®ҫзҪ®жҢүй’®ж ·ејҸгҖӮ

еҹәжң¬жҢүй’®ж ·ејҸ

е®һдҫӢ

.button {
  background-color: #4CAF50; /* Green */
  border: none;
  color: white;
  padding: 15px 32px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
}

дәІиҮӘиҜ•дёҖиҜ•

жҢүй’®йўңиүІ

иҜ·дҪҝз”Ё background-color еұһжҖ§жӣҙж”№жҢүй’®зҡ„иғҢжҷҜиүІпјҡ

е®һдҫӢ

.button1 {background-color: #4CAF50;} /* з»ҝиүІ */
.button2 {background-color: #008CBA;} /* и“қиүІ */
.button3 {background-color: #f44336;} /* зәўиүІ */ 
.button4 {background-color: #e7e7e7; color: black;} /* зҒ°иүІ */ 
.button5 {background-color: #555555;} /* й»‘иүІ */

дәІиҮӘиҜ•дёҖиҜ•

жҢүй’®е°әеҜё

иҜ·дҪҝз”Ё font-size еұһжҖ§жӣҙж”№жҢүй’®зҡ„еӯ—дҪ“еӨ§е°Ҹпјҡ

е®һдҫӢ

.button1 {font-size: 10px;}
.button2 {font-size: 12px;}
.button3 {font-size: 16px;}
.button4 {font-size: 20px;}
.button5 {font-size: 24px;}

дәІиҮӘиҜ•дёҖиҜ•

иҜ·дҪҝз”Ё padding еұһжҖ§жӣҙж”№жҢүй’®зҡ„еҶ…иҫ№и·қпјҡ

е®һдҫӢ

.button1 {padding: 10px 24px;}
.button2 {padding: 12px 28px;}
.button3 {padding: 14px 40px;}
.button4 {padding: 32px 16px;}
.button5 {padding: 16px;}

дәІиҮӘиҜ•дёҖиҜ•

еңҶи§’жҢүй’®

иҜ·дҪҝз”Ё border-radius еұһжҖ§дёәжҢүй’®ж·»еҠ еңҶи§’пјҡ

е®һдҫӢ

.button1 {border-radius: 2px;}
.button2 {border-radius: 4px;}
.button3 {border-radius: 8px;}
.button4 {border-radius: 12px;}

дәІиҮӘиҜ•дёҖиҜ•

иҜ·дҪҝз”Ё border еұһжҖ§дёәжҢүй’®ж·»еҠ еҪ©иүІиҫ№жЎҶпјҡ

е®һдҫӢ

.button1 {
  background-color: white;
  color: black;
  border: 2px solid #4CAF50; /* з»ҝиүІ */
}

дәІиҮӘиҜ•дёҖиҜ•

еҸҜжӮ¬еҒңжҢүй’®


еҪ“йј ж Үз§»еҠЁеҲ°жҢүй’®дёҠж–№ж—¶пјҢдҪҝз”Ё :hover йҖүжӢ©еҷЁеҸҜжӣҙж”№жҢүй’®зҡ„ж ·ејҸгҖӮ

жҸҗзӨәпјҡиҜ·дҪҝз”Ё transition-duration еұһжҖ§жқҘзЎ®е®ҡвҖңжӮ¬еҒңвҖқж•Ҳжһңзҡ„йҖҹеәҰпјҡ

е®һдҫӢ

.button {
  transition-duration: 0.4s;
}

.button:hover {
  background-color: #4CAF50; /* Green */
  color: white;
}

йҳҙеҪұжҢүй’®

иҜ·дҪҝз”Ё box-shadow еұһжҖ§дёәжҢүй’®ж·»еҠ йҳҙеҪұпјҡ

е®һдҫӢ

.button1 {
  box-shadow: 0 8px 16px 0 rgba(0,0,0,0.2), 0 6px 20px 0 rgba(0,0,0,0.19);
}

.button2:hover {
  box-shadow: 0 12px 16px 0 rgba(0,0,0,0.24), 0 17px 50px 0 rgba(0,0,0,0.19);
}

дәІиҮӘиҜ•дёҖиҜ•

зҰҒз”Ёзҡ„жҢүй’®

иҜ·дҪҝз”Ё opacity еұһжҖ§дёәжҢүй’®ж·»еҠ йҖҸжҳҺеәҰпјҲеҲӣе»әвҖңзҰҒз”ЁвҖқеӨ–и§ӮпјүгҖӮ

жҸҗзӨәпјҡжӮЁиҝҳеҸҜд»Ҙж·»еҠ еёҰжңү "not-allowed" еҖјзҡ„ cursor еұһжҖ§пјҢеҪ“жӮЁе°Ҷйј ж ҮжӮ¬еҒңеңЁжҢүй’®дёҠж—¶пјҢиҜҘеұһжҖ§дјҡжҳҫзӨә "no parking sign"пјҲзҰҒеҒңж Үеҝ—пјүпјҡ

е®һдҫӢ

.disabled {
  opacity: 0.6;
  cursor: not-allowed;
}

дәІиҮӘиҜ•дёҖиҜ•

жҢүй’®е®ҪеәҰ


й»ҳи®Өжғ…еҶөдёӢпјҢжҢүй’®зҡ„еӨ§е°ҸеҸ–еҶідәҺе…¶ж–Үжң¬еҶ…е®№пјҲдёҺеҶ…е®№зҡ„е®ҪеәҰдёҖж ·пјүгҖӮиҜ·дҪҝз”Ё width еұһжҖ§жқҘжӣҙж”№жҢүй’®зҡ„е®ҪеәҰпјҡ

е®һдҫӢ

.button1 {width: 250px;}
.button2 {width: 50%;}
.button3 {width: 100%;}

дәІиҮӘиҜ•дёҖиҜ•

жҢүй’®еҲҶз»„

 

еҲ йҷӨеӨ–иҫ№и·қе№¶еҗ‘жҜҸдёӘжҢүй’®ж·»еҠ  float:leftпјҢжқҘеҲӣе»әжҢүй’®з»„пјҡ

е®һдҫӢ

.button {
  float: left;
}

дәІиҮӘиҜ•дёҖиҜ•

е®һдҫӢ

.button {
  float: left;
  border: 1px solid green;
}

е®һдҫӢ

.button {
  display: block;
}
