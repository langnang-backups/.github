ж–ҮжЎЈзұ»еһӢе®ҡд№үпјҲDTDпјүеҸҜе®ҡд№үеҗҲжі•зҡ„XMLж–ҮжЎЈжһ„е»әжЁЎеқ—гҖӮе®ғдҪҝз”ЁдёҖзі»еҲ—еҗҲжі•зҡ„е…ғзҙ жқҘе®ҡд№үж–ҮжЎЈзҡ„з»“жһ„гҖӮ

DTD еҸҜиў«жҲҗиЎҢең°еЈ°жҳҺдәҺ XML ж–ҮжЎЈдёӯпјҢд№ҹеҸҜдҪңдёәдёҖдёӘеӨ–йғЁеј•з”ЁгҖӮ

еҶ…йғЁзҡ„ DOCTYPE еЈ°жҳҺ

еҒҮеҰӮ DTD иў«еҢ…еҗ«еңЁжӮЁзҡ„ XML жәҗж–Үд»¶дёӯпјҢе®ғеә”еҪ“йҖҡиҝҮдёӢйқўзҡ„иҜӯжі•еҢ…иЈ…еңЁдёҖдёӘ DOCTYPE еЈ°жҳҺдёӯпјҡ

<!DOCTYPE ж №е…ғзҙ  [е…ғзҙ еЈ°жҳҺ]>

еёҰжңү DTD зҡ„ XML ж–ҮжЎЈе®һдҫӢпјҲиҜ·еңЁ IE5 д»ҘеҸҠжӣҙй«ҳзҡ„зүҲжң¬жү“ејҖпјҢе№¶йҖүжӢ©жҹҘзңӢжәҗд»Јз Ғпјүпјҡ

<?xml version="1.0"?>
<!DOCTYPE note [
  <!ELEMENT note (to,from,heading,body)>
  <!ELEMENT to      (#PCDATA)>
  <!ELEMENT from    (#PCDATA)>
  <!ELEMENT heading (#PCDATA)>
  <!ELEMENT body    (#PCDATA)>
]>
<note>
  <to>George</to>
  <from>John</from>
  <heading>Reminder</heading>
  <body>Don't forget the meeting!</body>
</note>

еңЁжӮЁзҡ„жөҸи§ҲеҷЁдёӯжү“ејҖжӯӨ XML ж–Үд»¶пјҢе№¶йҖүжӢ©вҖңжҹҘзңӢжәҗд»Јз ҒвҖқе‘Ҫд»ӨгҖӮ

д»ҘдёҠ DTD и§ЈйҮҠеҰӮдёӢпјҡ

!DOCTYPE note (з¬¬дәҢиЎҢ)е®ҡд№үжӯӨж–ҮжЎЈжҳҜ note зұ»еһӢзҡ„ж–ҮжЎЈгҖӮ

!ELEMENT note (з¬¬дёүиЎҢ)е®ҡд№ү note е…ғзҙ жңүеӣӣдёӘе…ғзҙ пјҡ"toгҖҒfromгҖҒheading,гҖҒbody"

!ELEMENT to (з¬¬еӣӣиЎҢ)е®ҡд№ү to е…ғзҙ дёә "#PCDATA" зұ»еһӢ

!ELEMENT from (з¬¬дә”иЎҢ)е®ҡд№ү from е…ғзҙ дёә "#PCDATA" зұ»еһӢ

!ELEMENT heading (з¬¬е…ӯиЎҢ)е®ҡд№ү heading е…ғзҙ дёә "#PCDATA" зұ»еһӢ

!ELEMENT body (з¬¬дёғиЎҢ)е®ҡд№ү body е…ғзҙ дёә "#PCDATA" зұ»еһӢ

еӨ–йғЁж–ҮжЎЈеЈ°жҳҺ

еҒҮеҰӮ DTD дҪҚдәҺ XML жәҗж–Үд»¶зҡ„еӨ–йғЁпјҢйӮЈд№Ҳе®ғеә”йҖҡиҝҮдёӢйқўзҡ„иҜӯжі•иў«е°ҒиЈ…еңЁдёҖдёӘ DOCTYPE е®ҡд№үдёӯпјҡ

<!DOCTYPE ж №е…ғзҙ  SYSTEM "ж–Үд»¶еҗҚ">

иҝҷдёӘ XML ж–ҮжЎЈе’ҢдёҠйқўзҡ„ XML ж–ҮжЎЈзӣёеҗҢпјҢдҪҶжҳҜжӢҘжңүдёҖдёӘеӨ–йғЁзҡ„ DTD: пјҲеңЁ IE5 дёӯжү“ејҖпјҢе№¶йҖүжӢ©вҖңжҹҘзңӢжәҗд»Јз ҒвҖқе‘Ҫд»ӨгҖӮпјү

<?xml version="1.0"?>
<!DOCTYPE note SYSTEM "note.dtd">
<note>
<to>George</to>
<from>John</from>
<heading>Reminder</heading>
<body>Don't forget the meeting!</body>
</note>

<!ELEMENT note (to,from,heading,body)>

йҖҡиҝҮ DTDпјҢзӢ¬з«Ӣзҡ„еӣўдҪ“еҸҜдёҖиҮҙең°дҪҝз”ЁжҹҗдёӘж ҮеҮҶзҡ„ DTD жқҘдәӨжҚўж•°жҚ®гҖӮ
