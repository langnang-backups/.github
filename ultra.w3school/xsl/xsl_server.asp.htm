XSLT - еңЁжңҚеҠЎеҷЁдёҠ

  • XSLT еңЁе®ўжҲ·з«Ҝ
  • XSLT зј–иҫ‘ XML

з”ұдәҺе№¶йқһжүҖжңүзҡ„жөҸи§ҲеҷЁйғҪж”ҜжҢҒ XSLTпјҢеҸҰдёҖз§Қи§ЈеҶіж–№жЎҲжҳҜеңЁжңҚеҠЎеҷЁдёҠе®ҢжҲҗ XML иҮі XHTML зҡ„иҪ¬еҢ–гҖӮ

и·ЁжөҸи§ҲеҷЁи§ЈеҶіж–№жЎҲ

еңЁеүҚйқўзҡ„з« иҠӮпјҢжҲ‘д»¬и®Іи§ЈиҝҮеҰӮдҪ•еңЁжөҸи§ҲеҷЁдёӯдҪҝз”Ё XSLT жқҘе®ҢжҲҗ XML еҲ° XHTML зҡ„иҪ¬еҢ–гҖӮжҲ‘д»¬еҲӣе»әдәҶдёҖж®өдҪҝз”Ё XML и§ЈжһҗеҷЁжқҘиҝӣиЎҢиҪ¬еҢ–зҡ„ JavaScriptгҖӮJavaScript и§ЈеҶіж–№жЎҲж— жі•е·ҘдҪңдәҺжІЎжңү XML и§ЈжһҗеҷЁзҡ„жөҸи§ҲеҷЁгҖӮдёәдәҶи®© XML ж•°жҚ®йҖӮз”ЁдәҺд»»дҪ•зұ»еһӢзҡ„жөҸи§ҲеҷЁпјҢжҲ‘д»¬еҝ…йЎ»еңЁжңҚеҠЎеҷЁдёҠеҜ№ XML ж–ҮжЎЈиҝӣиЎҢиҪ¬жҚўпјҢз„¶еҗҺе°Ҷе…¶дҪңдёә XHMTL еҸ‘йҖҒеҲ°жөҸи§ҲеҷЁгҖӮ

иҝҷжҳҜ XSLT зҡ„еҸҰдёҖдёӘдјҳзӮ№гҖӮXSLT зҡ„и®ҫи®Ўзӣ®ж Үд№ӢдёҖжҳҜдҪҝж•°жҚ®еңЁжңҚеҠЎеҷЁдёҠд»ҺдёҖз§Қж јејҸиҪ¬жҚўеҲ°еҸҰдёҖз§Қж јејҸжҲҗдёәеҸҜиғҪпјҢе№¶еҗ‘жүҖжңүзұ»еһӢзҡ„жөҸи§ҲеҷЁиҝ”еӣһеҸҜиҜ»зҡ„ж•°жҚ®гҖӮ

XML ж–Үд»¶е’Ң XSL ж–Үд»¶

иҜ·зңӢиҝҷдёӘеңЁеүҚйқўзҡ„з« иҠӮе·Іеұ•зӨәиҝҮзҡ„ XML ж–ҮжЎЈпјҡ

<?xml version="1.0" encoding="ISO-8859-1"?>
<catalog>
  <cd>
    <title>Empire Burlesque</title>
    <artist>Bob Dylan</artist>
    <country>USA</country>
    <company>Columbia</company>
    <price>10.90</price>
    <year>1985</year>
  </cd>
.
.
.
</catalog>

жҹҘзңӢжӯӨ XML ж–Үд»¶гҖӮ

д»ҘеҸҠйҷ„йҡҸзҡ„ XSL ж ·ејҸиЎЁпјҡ

<?xml version="1.0" encoding="ISO-8859-1"?>
<xsl:stylesheet version="1.0"
xmlns:xsl="http://www.w3.org/1999/XSL/Transform">

<xsl:template match="/">
  <html>
  <body>
    <h2>My CD Collection</h2> 
    <table border="1">
      <tr bgcolor="#9acd32">
        <th align="left">Title</th> 
        <th align="left">Artist</th> 
      </tr>
      <xsl:for-each select="catalog/cd">
      <tr>
        <td><xsl:value-of select="title" /></td>
        <td><xsl:value-of select="artist" /></td>
      </tr>
      </xsl:for-each>
  </table>
  </body>
  </html>
</xsl:template>

</xsl:stylesheet>

жҹҘзңӢжӯӨ XSL ж–Үд»¶гҖӮ

иҜ·жіЁж„ҸпјҢиҝҷдёӘ XML ж–Үд»¶жІЎжңүеҢ…еҗ«еҜ№ XSL ж–Үд»¶зҡ„еј•з”ЁгҖӮ

йҮҚиҰҒдәӢйЎ№пјҡдёҠйқўиҝҷеҸҘиҜқиЎЁжҳҺпјҢXML ж–Үд»¶еҸҜдҪҝз”ЁеӨҡдёӘдёҚеҗҢзҡ„ XSL ж ·ејҸиЎЁжқҘиҝӣиЎҢиҪ¬жҚўгҖӮ

еңЁжңҚеҠЎеҷЁжҠҠ XML иҪ¬жҚўдёә XHTML

иҝҷжҳҜеңЁжңҚеҠЎеҷЁдёҠжҠҠ XML ж–Үд»¶иҪ¬жҚўдёә XHTML зҡ„жәҗд»Јз Ғпјҡ

<%
'Load XML
set xml = Server.CreateObject("Microsoft.XMLDOM")
xml.async = false
xml.load(Server.MapPath("cdcatalog.xml"))

'Load XSL
set xsl = Server.CreateObject("Microsoft.XMLDOM")
xsl.async = false
xsl.load(Server.MapPath("cdcatalog.xsl"))

'Transform file
Response.Write(xml.transformNode(xsl))
%>

жҸҗзӨәпјҡеҒҮеҰӮжӮЁдёҚдәҶи§ЈеҰӮдҪ•зј–еҶҷ ASPпјҢжӮЁеҸҜд»ҘеӯҰд№ жҲ‘д»¬зҡ„гҖҠASP ж•ҷзЁӢгҖӢгҖӮ

з¬¬дёҖж®өд»Јз ҒеҲӣе»әдәҶеҫ®иҪҜзҡ„ XML и§ЈжһҗеҷЁзҡ„дёҖдёӘе®һдҫӢпјҢз„¶еҗҺжҠҠ XML ж–Үд»¶иҪҪе…ҘдәҶеҶ…еӯҳгҖӮз¬¬дәҢж®өд»Јз ҒеҲӣе»әдәҶи§ЈжһҗеҷЁзҡ„еҸҰдёҖдёӘе®һдҫӢпјҢз„¶еҗҺжҠҠиҝҷдёӘ XSL ж–Үд»¶иҪҪе…ҘдәҶеҶ…еӯҳгҖӮжңҖеҗҺдёҖиЎҢд»Јз ҒдҪҝз”Ё XSL ж–ҮжЎЈиҪ¬жҚўдәҶ XML ж–ҮжЎЈпјҢе№¶жҠҠз»“жһңдҪңдёә XHTML еҸ‘йҖҒеҲ°жӮЁзҡ„жөҸи§ҲеҷЁгҖӮд»»еҠЎе®ҢжҲҗпјҒ
