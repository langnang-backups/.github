е®һдҫӢз ”з©¶пјҡеҰӮдҪ•дҪҝз”Ё XSLT е°Ҷ XML иҪ¬жҚўдёә XHTMLгҖӮ

жҲ‘д»¬дјҡеңЁдёӢдёҖиҠӮеҜ№жң¬дҫӢзҡ„з»ҶиҠӮиҝӣиЎҢи§ЈйҮҠгҖӮ

жӯЈзЎ®зҡ„ж ·ејҸиЎЁеЈ°жҳҺ

жҠҠж–ҮжЎЈеЈ°жҳҺдёә XSL ж ·ејҸиЎЁзҡ„ж №е…ғзҙ жҳҜ <xsl:stylesheet> жҲ– <xsl:transform>гҖӮ

жіЁйҮҠпјҡ <xsl:stylesheet> е’Ң <xsl:transform> жҳҜе®Ңе…ЁеҗҢд№үзҡ„пјҢеқҮеҸҜиў«дҪҝз”ЁпјҒ

ж №жҚ® W3C зҡ„ XSLT ж ҮеҮҶпјҢеЈ°жҳҺ XSL ж ·ејҸиЎЁзҡ„жӯЈзЎ®ж–№жі•жҳҜпјҡ

<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">

жҲ–иҖ…пјҡ

<xsl:transform version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">

еҰӮйңҖи®ҝй—® XSLT зҡ„е…ғзҙ гҖҒеұһжҖ§д»ҘеҸҠзү№жҖ§пјҢжҲ‘д»¬еҝ…йЎ»еңЁж–ҮжЎЈйЎ¶з«ҜеЈ°жҳҺ XSLT е‘ҪеҗҚз©әй—ҙгҖӮ

xmlns:xsl="http://www.w3.org/1999/XSL/Transform" жҢҮеҗ‘дәҶе®ҳж–№зҡ„ W3C XSLT е‘ҪеҗҚз©әй—ҙгҖӮеҰӮжһңжӮЁдҪҝз”ЁжӯӨе‘ҪеҗҚз©әй—ҙпјҢе°ұеҝ…йЎ»еҢ…еҗ«еұһжҖ§ version="1.0"гҖӮ

д»ҺдёҖдёӘеҺҹе§Ӣзҡ„ XML ж–ҮжЎЈејҖе§Ӣ

жҲ‘д»¬зҺ°еңЁиҰҒжҠҠдёӢйқўиҝҷдёӘ XML ж–ҮжЎЈпјҲ"cdcatalog.xml"пјүиҪ¬жҚўдёә XHTMLпјҡ

<?xml version="1.0" encoding="ISO-8859-1"?>
<catalog>
  <cd>
    <title>Empire Burlesque</title>
    <artist>Bob Dylan</artist>
    <country>USA</country>
    <company>Columbia</company>
    <price>10.90</price>
    <year>1985</year>
  </cd>
.
.
.
</catalog>

еңЁ Internet Explorer е’Ң Firefox дёӯжҹҘзңӢ XML ж–Үд»¶пјҡ

жү“ејҖ XML ж–Үд»¶пјҲйҖҡеёёйҖҡиҝҮзӮ№еҮ»жҹҗдёӘй“ҫжҺҘпјү - XML ж–ҮжЎЈдјҡд»ҘйўңиүІеҢ–зҡ„д»Јз Ғж–№ејҸжқҘжҳҫзӨәж №е…ғзҙ еҸҠеӯҗе…ғзҙ гҖӮзӮ№еҮ»е…ғзҙ е·Ұдҫ§зҡ„еҠ еҸ·жҲ–еҮҸеҸ·еҸҜеұ•ејҖжҲ–ж”¶зј©е…ғзҙ зҡ„з»“жһ„гҖӮеҰӮйңҖжҹҘзңӢеҺҹе§Ӣзҡ„XMLжәҗж–Үд»¶пјҲдёҚеёҰжңүеҠ еҸ·е’ҢеҮҸеҸ·пјүпјҢиҜ·еңЁжөҸи§ҲеҷЁиҸңеҚ•дёӯйҖүжӢ©вҖңжҹҘзңӢйЎөйқўжәҗд»Јз ҒвҖқгҖӮ

еңЁ Netscape 6 дёӯжҹҘзңӢ XML ж–Үд»¶пјҡ

жү“ејҖ XML ж–Үд»¶пјҢз„¶еҗҺеңЁ XML ж–Үд»¶дёӯеҸіеҮ»пјҢе№¶йҖүжӢ©вҖңжҹҘзңӢйЎөйқўжәҗд»Јз ҒвҖқгҖӮXMLж–ҮжЎЈдјҡд»ҘйўңиүІеҢ–зҡ„д»Јз Ғж–№ејҸжқҘжҳҫзӨәж №е…ғзҙ еҸҠеӯҗе…ғзҙ гҖӮ

еңЁ Opera 7 дёӯжҹҘзңӢ XML ж–Үд»¶пјҡ

жү“ејҖ XML ж–Үд»¶пјҢз„¶еҗҺеңЁXMLж–Үд»¶дёӯеҸіеҮ»пјҢйҖүжӢ©вҖңжЎҶжһ¶вҖқ/вҖңжҹҘзңӢжәҗд»Јз ҒвҖқгҖӮXMLж–ҮжЎЈе°ҶжҳҫзӨәдёәзәҜж–Үжң¬гҖӮ

жҹҘзңӢ"cdcatalog.xml"гҖӮ

еҲӣе»ә XSL ж ·ејҸиЎЁ

з„¶еҗҺеҲӣе»әдёҖдёӘеёҰжңүиҪ¬жҚўжЁЎжқҝзҡ„ XSL ж ·ејҸиЎЁпјҲ"cdcatalog.xsl"пјүпјҡ

<?xml version="1.0" encoding="ISO-8859-1"?>

<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">

<xsl:template match="/">
  <html>
  <body>
    <h2>My CD Collection</h2>
    <table border="1">
    <tr bgcolor="#9acd32">
      <th align="left">Title</th>
      <th align="left">Artist</th>
    </tr>
    <xsl:for-each select="catalog/cd">
    <tr>
      <td><xsl:value-of select="title"/></td>
      <td><xsl:value-of select="artist"/></td>
    </tr>
    </xsl:for-each>
    </table>
  </body>
  </html>
</xsl:template>

</xsl:stylesheet>
