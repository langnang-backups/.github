PHP иЎЁеҚ•йӘҢиҜҒ

еӨ„зҗҶ PHP иЎЁеҚ•ж—¶иҜ·иҖғиҷ‘е®үе…ЁжҖ§пјҒ

иҝҷдәӣйЎөйқўе°Ҷеұ•зӨәеҰӮдҪ•еңЁзЎ®дҝқе®үе…ЁжҖ§зҡ„еүҚжҸҗдёӢеӨ„зҗҶ PHP иЎЁеҚ•гҖӮйҖӮеҪ“зҡ„иЎЁеҚ•ж•°жҚ®йӘҢиҜҒеҜ№дәҺдҝқжҠӨиЎЁеҚ•е…ҚеҸ—й»‘е®ўе’ҢеһғеңҫйӮ®д»¶еҸ‘йҖҒиҖ…зҡ„ж”»еҮ»йқһеёёйҮҚиҰҒпјҒ

жҲ‘д»¬е°ҶеңЁиҝҷдәӣз« иҠӮдёӯеҲ¶дҪңзҡ„ HTML иЎЁеҚ•дёӯеҢ…еҗ«еҗ„з§Қиҫ“е…Ҙеӯ—ж®өпјҡеҝ…еЎ«е’ҢеҸҜйҖүзҡ„ж–Үжң¬еӯ—ж®өгҖҒеҚ•йҖүжҢүй’®е’ҢдёҖдёӘжҸҗдәӨжҢүй’®пјҡ

дёҠиҝ°иЎЁеҚ•зҡ„йӘҢиҜҒи§„еҲҷеҰӮдёӢпјҡ

еӯ—ж®ө йӘҢиҜҒи§„еҲҷ
е§“еҗҚ еҝ…еЎ«гҖӮд»…еҢ…еҗ«еӯ—жҜҚе’Ңз©әж јгҖӮ
з”өеӯҗйӮ®д»¶ еҝ…еЎ«гҖӮеҝ…йЎ»еҢ…еҗ«жңүж•Ҳзҡ„з”өеӯҗйӮ®д»¶ең°еқҖпјҲеёҰжңү @ е’Ң .пјүгҖӮ
зҪ‘з«ҷ еҸҜйҖүгҖӮеҰӮжһңеӯҳеңЁпјҢеҝ…йЎ»еҢ…еҗ«жңүж•Ҳзҡ„ URLгҖӮ
иҜ„и®ә еҸҜйҖүгҖӮеӨҡиЎҢиҫ“е…Ҙеӯ—ж®өпјҲtextareaпјүгҖӮ
жҖ§еҲ« еҝ…еЎ«гҖӮеҝ…йЎ»йҖүжӢ©дёҖдёӘгҖӮ

йҰ–е…ҲпјҢжҲ‘д»¬е°ҶжҹҘзңӢиЎЁеҚ•зҡ„зәҜ HTML д»Јз Ғпјҡ

ж–Үжң¬еӯ—ж®ө

е§“еҗҚгҖҒз”өеӯҗйӮ®д»¶е’ҢзҪ‘з«ҷеӯ—ж®өжҳҜж–Үжң¬иҫ“е…Ҙе…ғзҙ пјҢиҖҢиҜ„и®әеӯ—ж®өжҳҜж–Үжң¬еҢәеҹҹгҖӮ

HTML д»Јз ҒзңӢиө·жқҘеғҸиҝҷж ·пјҡ

е§“еҗҚпјҡ<input type="text" name="name">
з”өйӮ®пјҡ<input type="text" name="email">
зҪ‘з«ҷпјҡ<input type="text" name="website">
иҜ„и®әпјҡ<textarea name="comment" rows="5" cols="40"></textarea>

еҚ•йҖүжҢүй’®

жҖ§еҲ«еӯ—ж®өжҳҜеҚ•йҖүжҢүй’®пјҢHTML д»Јз ҒзңӢиө·жқҘеғҸиҝҷж ·пјҡ

жҖ§еҲ«пјҡ
<input type="radio" name="gender" value="female">еҘіжҖ§
<input type="radio" name="gender" value="male">з”·жҖ§
<input type="radio" name="gender" value="other">е…¶д»–

иЎЁеҚ•е…ғзҙ 

иЎЁеҚ•зҡ„ HTML д»Јз ҒзңӢиө·жқҘеғҸиҝҷж ·пјҡ

<form method="post" action="<?php echo htmlspecialchars($_SERVER["PHP_SELF"]);?>">

еҪ“иЎЁеҚ•жҸҗдәӨж—¶пјҢиЎЁеҚ•ж•°жҚ®йҖҡиҝҮ method="post" еҸ‘йҖҒгҖӮ

д»Җд№ҲжҳҜ $_SERVER["PHP_SELF"] еҸҳйҮҸпјҹ

$_SERVER["PHP_SELF"] жҳҜдёҖдёӘи¶…е…ЁеұҖеҸҳйҮҸпјҢиҝ”еӣһеҪ“еүҚжү§иЎҢи„ҡжң¬зҡ„ж–Үд»¶еҗҚгҖӮ

еӣ жӯӨпјҢ$_SERVER["PHP_SELF"] е°ҶжҸҗдәӨзҡ„иЎЁеҚ•ж•°жҚ®еҸ‘йҖҒеӣһиҮӘиә«йЎөйқўпјҢиҖҢдёҚжҳҜи·іиҪ¬еҲ°еҸҰдёҖдёӘйЎөйқўгҖӮиҝҷж ·пјҢз”ЁжҲ·е°ҶеңЁдёҺиЎЁеҚ•зӣёеҗҢзҡ„йЎөйқўдёҠж”¶еҲ°й”ҷиҜҜж¶ҲжҒҜгҖӮ

д»Җд№ҲжҳҜ htmlspecialchars() еҮҪж•°пјҹ

htmlspecialchars() еҮҪж•°е°Ҷзү№ж®Ҡеӯ—з¬ҰиҪ¬жҚўдёә HTML е®һдҪ“гҖӮиҝҷж„Ҹе‘ізқҖе®ғе°ҶжӣҝжҚў HTML еӯ—з¬ҰпјҢеҰӮ < е’Ң >пјҢдёә < е’Ң >гҖӮиҝҷеҸҜд»ҘйҳІжӯўж”»еҮ»иҖ…йҖҡиҝҮеңЁиЎЁеҚ•дёӯжіЁе…Ҙ HTML жҲ– JavaScript д»Јз ҒпјҲи·Ёз«ҷи„ҡжң¬ж”»еҮ»пјүжқҘеҲ©з”Ёд»Јз ҒгҖӮ

иӯҰе‘ҠпјҒ

$_SERVER["PHP_SELF"] еҸҳйҮҸеҸҜиғҪиў«й»‘е®ўеҲ©з”ЁпјҒ

еҰӮжһңеңЁжӮЁзҡ„йЎөйқўдёӯдҪҝз”Ё PHP_SELFпјҢеҲҷз”ЁжҲ·еҸҜд»Ҙиҫ“е…ҘдёҖдёӘж–ңжқ  /пјҢз„¶еҗҺиҫ“е…ҘдёҖдәӣи·Ёз«ҷи„ҡжң¬пјҲXSSпјүе‘Ҫд»ӨжқҘжү§иЎҢгҖӮ

и·Ёз«ҷи„ҡжң¬пјҲXSSпјүжҳҜдёҖз§ҚйҖҡеёёеңЁ Web еә”з”ЁзЁӢеәҸдёӯеҸ‘зҺ°зҡ„и®Ўз®—жңәе®үе…ЁжјҸжҙһгҖӮXSS дҪҝж”»еҮ»иҖ…иғҪеӨҹе°Ҷе®ўжҲ·з«Ҝи„ҡжң¬жіЁе…ҘеҲ°з”ұе…¶д»–з”ЁжҲ·жҹҘзңӢзҡ„ Web йЎөйқўдёӯгҖӮ

еҒҮи®ҫжҲ‘д»¬еңЁеҗҚдёә "test_form.php" зҡ„йЎөйқўдёӯжңүд»ҘдёӢиЎЁеҚ•пјҡ

<form method="post" action="<?php echo $_SERVER["PHP_SELF"];?>">

зҺ°еңЁпјҢеҰӮжһңз”ЁжҲ·еңЁең°еқҖж Ҹдёӯиҫ“е…ҘжӯЈеёёзҡ„ URLпјҢеҰӮ "http://www.example.com/test_form.php"пјҢеҲҷдёҠиҝ°д»Јз Ғе°Ҷиў«зҝ»иҜ‘дёәпјҡ

<form method="post" action="test_form.php">

еҲ°зӣ®еүҚдёәжӯўпјҢдёҖеҲҮжӯЈеёёгҖӮ

дҪҶжҳҜпјҢиҜ·жҖқиҖғз”ЁжҲ·еңЁең°еқҖж Ҹдёӯиҫ“е…Ҙд»ҘдёӢ URLпјҡ

http://www.example.com/test_form.php/%22%3E%3Cscript%3Ealert('hacked')%3C/script%3E

еңЁиҝҷз§Қжғ…еҶөдёӢпјҢдёҠиҝ°д»Јз Ғе°Ҷиў«зҝ»иҜ‘дёәпјҡ

<form method="post" action="test_form.php/"><script>alert('hacked')</script>

жӯӨд»Јз Ғж·»еҠ дәҶдёҖдёӘи„ҡжң¬ж Үзӯҫе’ҢдёҖдёӘ alert е‘Ҫд»ӨгҖӮеҪ“йЎөйқўеҠ иҪҪж—¶пјҢJavaScript д»Јз Ғе°Ҷиў«жү§иЎҢпјҲз”ЁжҲ·е°ҶзңӢеҲ°дёҖдёӘиӯҰе‘ҠжЎҶпјүгҖӮиҝҷеҸӘжҳҜдёҖдёӘз®ҖеҚ•зҡ„ж— е®ізӨәдҫӢпјҢеұ•зӨәдәҶеҰӮдҪ•еҲ©з”Ё PHP_SELF еҸҳйҮҸгҖӮ

иҜ·жіЁж„ҸпјҢеҸҜд»ҘеңЁ <script> ж ҮзӯҫеҶ…ж·»еҠ д»»дҪ• JavaScript д»Јз ҒпјҒй»‘е®ўеҸҜд»Ҙе°Ҷз”ЁжҲ·йҮҚе®ҡеҗ‘еҲ°еҸҰдёҖеҸ°жңҚеҠЎеҷЁдёҠзҡ„ж–Үд»¶пјҢиҖҢиҜҘж–Үд»¶еҸҜиғҪеҢ…еҗ«еҸҜд»Ҙжӣҙж”№е…ЁеұҖеҸҳйҮҸжҲ–е°ҶиЎЁеҚ•жҸҗдәӨеҲ°еҸҰдёҖдёӘең°еқҖд»Ҙдҝқеӯҳз”ЁжҲ·ж•°жҚ®зҡ„жҒ¶ж„Ҹд»Јз ҒгҖӮ

еҰӮдҪ•йҒҝе…Қ $_SERVER["PHP_SELF"] иў«жҒ¶ж„ҸдҪҝз”Ёпјҹ

йҖҡиҝҮдҪҝз”Ё htmlspecialchars() еҮҪж•°еҸҜд»ҘйҒҝе…Қ $_SERVER["PHP_SELF"] иў«жҒ¶ж„ҸдҪҝз”ЁгҖӮ

иЎЁеҚ•д»Јз Ғеә”иҜҘеғҸиҝҷж ·пјҡ

<form method="post" action="<?php echo htmlspecialchars($_SERVER["PHP_SELF"]);?>">

htmlspecialchars() еҮҪж•°е°Ҷзү№ж®Ҡеӯ—з¬ҰиҪ¬жҚўдёә HTML е®һдҪ“гҖӮзҺ°еңЁпјҢеҰӮжһңз”ЁжҲ·е°қиҜ•еҲ©з”Ё PHP_SELF еҸҳйҮҸпјҢз»“жһңе°ҶеҰӮдёӢжүҖзӨәпјҡ

<form method="post" action="test_form.php/&quot;&gt;&lt;script&gt;alert('hacked')&lt;/script&gt;">

жҒ¶ж„ҸдҪҝз”Ёзҡ„е°қиҜ•еӨұиҙҘпјҢжІЎжңүйҖ жҲҗд»»дҪ•жҚҹе®іпјҒ

дҪҝз”Ё PHP йӘҢиҜҒиЎЁеҚ•ж•°жҚ®

жҲ‘д»¬иҰҒеҒҡзҡ„з¬¬дёҖд»¶дәӢжҳҜе°ҶжүҖжңүеҸҳйҮҸйҖҡиҝҮ PHP зҡ„ htmlspecialchars() еҮҪж•°дј йҖ’гҖӮ

еҪ“жҲ‘д»¬дҪҝз”Ё htmlspecialchars() еҮҪж•°ж—¶пјӣеҰӮжһңз”ЁжҲ·е°қиҜ•еңЁж–Үжң¬еӯ—ж®өдёӯжҸҗдәӨд»ҘдёӢеҶ…е®№пјҡ

<script>location.href('http://www.hacked.com')</script>

е®ғдёҚдјҡиў«жү§иЎҢпјҢеӣ дёәе®ғе°Ҷиў«дҝқеӯҳдёә HTML иҪ¬д№үд»Јз ҒпјҢеҰӮдёӢжүҖзӨәпјҡ

&lt;script&gt;location.href('http://www.hacked.com')&lt;/script&gt;

зҺ°еңЁд»Јз ҒеҸҜд»Ҙе®үе…Ёең°жҳҫзӨәеңЁйЎөйқўдёҠжҲ–з”өеӯҗйӮ®д»¶дёӯгҖӮ

еҪ“з”ЁжҲ·жҸҗдәӨиЎЁеҚ•ж—¶пјҢжҲ‘д»¬иҝҳе°Ҷжү§иЎҢеҸҰеӨ–дёӨйЎ№ж“ҚдҪңпјҡ

  • дҪҝз”Ё PHP trim() еҮҪж•°д»Һз”ЁжҲ·иҫ“е…Ҙж•°жҚ®дёӯеҲ йҷӨдёҚеҝ…иҰҒзҡ„еӯ—з¬ҰпјҲйўқеӨ–зҡ„з©әж јгҖҒеҲ¶иЎЁз¬ҰгҖҒжҚўиЎҢз¬Ұпјү
  • дҪҝз”Ё PHP stripslashes() еҮҪж•°д»Һз”ЁжҲ·иҫ“е…Ҙж•°жҚ®дёӯеҲ йҷӨеҸҚж–ңжқ  \

дёӢдёҖжӯҘжҳҜеҲӣе»әдёҖдёӘеҮҪж•°жқҘдёәжҲ‘д»¬жү§иЎҢжүҖжңүжЈҖжҹҘпјҲиҝҷжҜ”дёҖйҒҚеҸҲдёҖйҒҚең°зј–еҶҷзӣёеҗҢзҡ„д»Јз ҒиҰҒж–№дҫҝеҫ—еӨҡпјүгҖӮ

жҲ‘д»¬е°ҶиҜҘеҮҪж•°е‘ҪеҗҚдёә test_input()гҖӮ

зҺ°еңЁпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ё test_input() еҮҪж•°жЈҖжҹҘжҜҸдёӘ $_POST еҸҳйҮҸпјҢи„ҡжң¬еҰӮдёӢжүҖзӨәпјҡ

е®һдҫӢ

// е®ҡд№үеҸҳйҮҸе№¶и®ҫзҪ®дёәз©әеҖј
$name = $email = $gender = $comment = $website = "";

if ($_SERVER["REQUEST_METHOD"] == "POST") {
  $name = test_input($_POST["name"]);
  $email = test_input($_POST["email"]);
  $website = test_input($_POST["website"]);
  $comment = test_input($_POST["comment"]);
  $gender = test_input($_POST["gender"]);
}


function test_input($data) {
  $data = trim($data);
  $data = stripslashes($data);
  $data = htmlspecialchars($data);
  return $data;
}
