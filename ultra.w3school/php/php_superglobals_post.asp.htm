PHP $_POST

$_POST еҢ…еҗ«йҖҡиҝҮ HTTP POST ж–№жі•жҺҘж”¶зҡ„еҸҳйҮҸж•°з»„гҖӮ

йҖҡиҝҮ HTTP Post ж–№жі•еҸ‘йҖҒеҸҳйҮҸзҡ„дё»иҰҒжңүдёӨз§Қж–№ејҸпјҡ

  • HTML иЎЁеҚ•
  • JavaScript HTTP иҜ·жұӮ

HTML иЎЁеҚ•дёӯзҡ„ $_POST

еҰӮжһң HTML иЎЁеҚ•зҡ„ method еұһжҖ§и®ҫзҪ®дёә "POST"пјҢеҲҷиҜҘиЎЁеҚ•дјҡйҖҡиҝҮ HTTP POST ж–№жі•жҸҗдәӨдҝЎжҒҜгҖӮ

дёәдәҶжј”зӨәиҝҷдёҖзӮ№пјҢжҲ‘д»¬йҰ–е…ҲеҲӣе»әдёҖдёӘз®ҖеҚ•зҡ„ HTML иЎЁеҚ•пјҡ

HTML иЎЁеҚ•

<html>
<body>

<form method="POST" action="demo_request.php">
  Name: <input type="text" name="fname">
  <input type="submit">
</form>

</body>
</html>

еҪ“з”ЁжҲ·зӮ№еҮ»жҸҗдәӨжҢүй’®ж—¶пјҢиЎЁеҚ•ж•°жҚ®дјҡиў«еҸ‘йҖҒеҲ° <form> ж Үзӯҫзҡ„ action еұһжҖ§дёӯжҢҮе®ҡзҡ„ PHP ж–Үд»¶гҖӮ

еңЁ action ж–Үд»¶дёӯпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ё $_POST еҸҳйҮҸжқҘж”¶йӣҶиҫ“е…Ҙеӯ—ж®өзҡ„еҖјгҖӮ

PHP ж–Үд»¶

$name = $_POST['fname'];
echo $name;

еңЁдёӢдҫӢдёӯпјҢжҲ‘д»¬е°Ҷ HTML иЎЁеҚ•е’Ң PHP д»Јз Ғж”ҫеңЁдәҶеҗҢдёҖдёӘ PHP ж–Үд»¶дёӯгҖӮ

жҲ‘д»¬иҝҳж·»еҠ дәҶдёҖдәӣйўқеӨ–зҡ„е®үе…ЁиЎҢгҖӮ

JavaScript HTTP иҜ·жұӮдёӯзҡ„ $_POST

еңЁ JavaScript дёӯеҸ‘йҖҒ HTTP иҜ·жұӮж—¶пјҢжӮЁеҸҜд»ҘжҢҮе®ҡ HTTP ж–№жі•дёә POSTгҖӮ

дёәдәҶжј”зӨәиҝҷдёҖзӮ№пјҢжҲ‘д»¬йҰ–е…ҲеҲӣе»әдёҖдёӘеҢ…еҗ« HTTP иҜ·жұӮзҡ„ JavaScript еҮҪж•°пјҡ

JavaScript еҮҪж•°

function myfunction() {
  const xhttp = new XMLHttpRequest();
  xhttp.open("POST", "demo_phpfile.php");
  xhttp.setRequestHeader("Content-type", "application/x-www-form-urlencoded");
  xhttp.onload = function() {
    document.getElementById("demo").innerHTML = this.responseText;
  }
  xhttp.send("fname=Mary");
  }
}

дёҠйқўзҡ„д»Јз Ғдјҡпјҡ

  • еҸ‘иө·дёҖдёӘ HTTP иҜ·жұӮ
  • и®ҫзҪ® HTTP ж–№жі•дёә POST
  • и®ҫзҪ®жңүж•Ҳзҡ„иҜ·жұӮеӨҙ
  • еҲӣе»әдёҖдёӘеңЁиҜ·жұӮе®ҢжҲҗж—¶жү§иЎҢзҡ„еҮҪж•°
  • еҸ‘йҖҒ HTTP иҜ·жұӮпјҢе№¶е°ҶеҸҳйҮҸ fname и®ҫзҪ®дёә Mary

жҹҘзңӢеҪ“иҜ·жұӮе®ҢжҲҗж—¶жү§иЎҢзҡ„еҮҪж•°пјҡ

xhttp.onload = function() {
    document.getElementById("demo").innerHTML = this.responseText;
  }

е®ғдјҡе°қиҜ•е°ҶеңЁж“ҚдҪңдёӯеҫ—еҲ°зҡ„е“Қеә”еҶҷе…Ҙе…·жңү id="demo" зҡ„ HTML е…ғзҙ гҖӮ

и®©жҲ‘д»¬еҲӣе»әдёҖдёӘеёҰжңүжӯӨзұ»е…ғзҙ зҡ„ HTML йЎөйқўпјҢд»ҘеҸҠдёҖдёӘжү§иЎҢиҜҘеҮҪж•°зҡ„жҢүй’®гҖӮ

еҰӮжһңжҲ‘д»¬иҝҳж·»еҠ дәҶ JavaScriptпјҢеҲҷйЎөйқўзңӢиө·жқҘеғҸиҝҷж ·пјҡ

е®һдҫӢ
