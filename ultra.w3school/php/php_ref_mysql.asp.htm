PHP MySQL з®Җд»Ӣ

MySQL еҮҪж•°е…Ғи®ёжӮЁи®ҝй—® MySQL ж•°жҚ®еә“жңҚеҠЎеҷЁгҖӮ

е®үиЈ…

дёәдәҶиғҪеӨҹйЎәеҲ©зҡ„дҪҝз”Ёжң¬зұ»еҮҪж•°пјҢеҝ…йЎ»еңЁзј–иҜ‘ PHP ж—¶ж·»еҠ еҜ№ MySQL зҡ„ж”ҜжҢҒгҖӮ

зј–иҜ‘ж—¶пјҢеҸӘиҰҒдҪҝз”Ё --with-mysql[=DIR] й…ҚзҪ®йҖүйЎ№еҚіеҸҜпјҢе…¶дёӯеҸҜйҖүзҡ„ [DIR] жҢҮеҗ‘ MySQL зҡ„е®үиЈ…зӣ®еҪ•гҖӮ

иҷҪз„¶жң¬ MySQL жү©еұ•еә“е…је®№ MySQL 4.1.0 еҸҠе…¶д»ҘеҗҺзүҲжң¬пјҢдҪҶжҳҜе®ғдёҚж”ҜжҢҒиҝҷдәӣзүҲжң¬жҸҗдҫӣзҡ„йўқеӨ–еҠҹиғҪгҖӮиҰҒдҪҝз”ЁиҝҷдәӣеҠҹиғҪпјҢиҜ·дҪҝз”Ё MySQLi жү©еұ•еә“гҖӮ

еҰӮжһңиҰҒеҗҢж—¶е®үиЈ… mysql жү©еұ•еә“е’Ң mysqli жү©еұ•еә“пјҢеҝ…йЎ»дҪҝз”ЁеҗҢдёҖдёӘе®ўжҲ·з«Ҝеә“д»ҘйҒҝе…Қд»»дҪ•еҶІзӘҒгҖӮ

еңЁ Linux зі»з»ҹдёҠе®үиЈ…

PHP 4

й»ҳи®ӨејҖеҗҜдәҶ --with-mysql йҖүйЎ№гҖӮжӯӨй»ҳи®ӨиЎҢдёәеҸҜд»Ҙз”Ё --without-mysql й…ҚзҪ®йҖүйЎ№жқҘзҰҒжӯўгҖӮеҰӮжһңеҗҜз”Ё MySQL иҖҢдёҚжҢҮе®ҡе®үиЈ…зӣ®еҪ•зҡ„иҜқпјҢPHP е°ҶдҪҝз”Ёз»‘е®ҡзҡ„ MySQL е®ўжҲ·з«ҜиҝһжҺҘеә“гҖӮ

иҝҳжңүе…¶е®ғеә”з”ЁзЁӢеәҸдҪҝз”Ё MySQLпјҲдҫӢеҰӮ auth-mysqlпјүзҡ„з”ЁжҲ·дёҚиҰҒз”Ёз»‘е®ҡзҡ„еә“пјҢиҖҢиҰҒжҢҮе®ҡ MySQL зҡ„е®үиЈ…зӣ®еҪ•пјҢеҰӮиҝҷж ·пјҡ--with-mysql=/path/to/mysqlгҖӮиҝҷе°ҶејәеҲ¶ PHP дҪҝз”ЁйҡҸ MySQL е®үиЈ…зҡ„е®ўжҲ·з«ҜиҝһжҺҘеә“пјҢе°ұеҸҜд»ҘйҒҝе…Қд»»дҪ•еҶІзӘҒгҖӮ

PHP 5+

MySQL й»ҳи®ӨжңӘеҗҜз”ЁпјҢд№ҹжІЎжңүз»‘е®ҡзҡ„ MySQL еә“гҖӮдҪҝз”Ё --with-mysql[=DIR] й…ҚзҪ®йҖүйЎ№жқҘеҠ е…Ҙ MySQL зҡ„ж”ҜжҢҒгҖӮеҸҜд»Ҙд»Һ MySQL дёӢиҪҪеӨҙж–Үд»¶е’Ңеә“гҖӮ

еңЁ Windows зі»з»ҹдёҠе®үиЈ…

PHP 4

PHP MySQL жү©еұ•е·Із»Ҹзј–иҜ‘е…Ҙ PHPгҖӮ

PHP 5+

MySQL й»ҳи®ӨжңӘеҗҜз”ЁпјҢеӣ жӯӨеҝ…йЎ»еңЁ php.ini дёӯжҝҖжҙ» php_mysql.dll еҠЁжҖҒиҝһжҺҘеә“гҖӮжӯӨеӨ–пјҢPHP иҝҳйңҖиҰҒи®ҝй—® MySQL е®ўжҲ·з«ҜиҝһжҺҘеә“гҖӮPHP зҡ„ Windows еҸ‘иЎҢзүҲеҢ…жӢ¬дәҶдёҖдёӘ libmysql.dllпјҢдёәдәҶи®© PHP иғҪе’Ң MySQL еҜ№иҜқпјҢжӯӨж–Үд»¶еҝ…йЎ»ж”ҫеңЁ Windows зҡ„зі»з»ҹи·Ҝеҫ„ PATH дёӯгҖӮ

иҰҒжҝҖжҙ»д»»дҪ• PHP жү©еұ•еә“пјҲдҫӢеҰӮ php_mysql.dllпјүпјҢPHP жҢҮд»Ө extension_dir иҰҒиў«и®ҫдёә PHP жү©еұ•еә“жүҖеңЁзҡ„зӣ®еҪ•гҖӮPHP 5 дёӢ extension_dir еҸ–еҖјзҡ„дёҖдёӘдҫӢеӯҗжҳҜ c:\php\extгҖӮ

жіЁйҮҠпјҡеҰӮжһңеҗҜеҠЁ web жңҚеҠЎеҷЁж—¶еҮәзҺ°зұ»дјјеҰӮдёӢзҡ„й”ҷиҜҜпјҡ"Unable to load dynamic library './php_mysql.dll'"пјҢиҝҷжҳҜеӣ дёәзі»з»ҹжүҫдёҚеҲ° php_mysql.dll е’Ң / жҲ– libmysql.dllгҖӮ

Runtime й…ҚзҪ®

MySQL еҮҪж•°зҡ„иЎҢдёәеҸ—еҲ° php.ini дёӯи®ҫзҪ®зҡ„еҪұе“ҚгҖӮ

MySQL й…ҚзҪ®йҖүйЎ№пјҡ

еҗҚз§° й»ҳи®Ө жҸҸиҝ° еҸҜжӣҙж”№
mysql.allow_persistent "1" жҳҜеҗҰе…Ғи®ё MySQL зҡ„жҢҒд№…иҝһжҺҘгҖӮ PHP_INI_SYSTEM
mysql.max_persistent "-1" жҜҸдёӘиҝӣзЁӢдёӯжңҖеӨ§зҡ„жҢҒд№…иҝһжҺҘж•°зӣ®гҖӮ PHP_INI_SYSTEM
mysql.max_links "-1" жҜҸдёӘиҝӣзЁӢдёӯжңҖеӨ§зҡ„иҝһжҺҘж•°пјҢеҢ…жӢ¬жҢҒд№…иҝһжҺҘгҖӮ PHP_INI_SYSTEM
mysql.trace_mode "0" и·ҹиёӘжЁЎејҸгҖӮд»Һ PHP 4.3.0 иө·еҸҜз”ЁгҖӮ PHP_INI_ALL
mysql.default_port NULL жҢҮе®ҡй»ҳи®ӨиҝһжҺҘж•°жҚ®еә“зҡ„ TCP з«ҜеҸЈеҸ·гҖӮ PHP_INI_ALL
mysql.default_socket NULL й»ҳи®Өзҡ„ socket еҗҚз§°гҖӮPHP 4.0.1иө·еҸҜз”ЁгҖӮ PHP_INI_ALL
mysql.default_host NULL й»ҳи®Өзҡ„жңҚеҠЎеҷЁең°еқҖгҖӮдёҚйҖӮз”ЁдәҺSQLе®үе…ЁжЁЎејҸгҖӮ PHP_INI_ALL
mysql.default_user NULL й»ҳи®ӨдҪҝз”Ёзҡ„з”ЁжҲ·еҗҚгҖӮдёҚйҖӮз”ЁдәҺSQLе®үе…ЁжЁЎејҸгҖӮ PHP_INI_ALL
mysql.default_password NULL й»ҳи®ӨдҪҝз”Ёзҡ„еҜҶз ҒгҖӮдёҚйҖӮз”ЁдәҺSQLе®үе…ЁжЁЎејҸгҖӮ PHP_INI_ALL
mysql.connect_timeout "60" иҝһжҺҘи¶…ж—¶з§’ж•°гҖӮ PHP_INI_ALL

еңЁ MySQL жЁЎеқ—дёӯдҪҝз”ЁдәҶдёӨз§Қиө„жәҗзұ»еһӢгҖӮз¬¬дёҖз§ҚжҳҜж•°жҚ®еә“зҡ„иҝһжҺҘеҸҘжҹ„пјҢз¬¬дәҢз§ҚжҳҜ SQL жҹҘиҜўиҝ”еӣһзҡ„з»“жһңйӣҶгҖӮ

PHP MySQL еҮҪж•°

еҮҪж•° жҸҸиҝ°
mysql_affected_rows() еҸ–еҫ—еүҚдёҖж¬Ў MySQL ж“ҚдҪңжүҖеҪұе“Қзҡ„и®°еҪ•иЎҢж•°гҖӮ
mysql_change_user() дёҚиөһжҲҗгҖӮж”№еҸҳжҙ»еҠЁиҝһжҺҘдёӯзҷ»еҪ•зҡ„з”ЁжҲ·
mysql_client_encoding() иҝ”еӣһеҪ“еүҚиҝһжҺҘзҡ„еӯ—з¬ҰйӣҶзҡ„еҗҚз§°

дҪҝз”Ё mysql_query() д»ЈжӣҝгҖӮ
