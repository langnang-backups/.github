JavaScript ж—Ҙжңҹи®ҫзҪ®ж–№жі•

  • JS ж—ҘжңҹиҺ·еҸ–ж–№жі•
  • JS ж•°еӯҰ

дҪҝз”ЁвҖңи®ҫзҪ®ж—ҘжңҹвҖқж–№жі•еҸҜд»Ҙи®ҫзҪ®ж—ҘжңҹеҜ№иұЎзҡ„ж—ҘжңҹеҖјпјҲе№ҙгҖҒжңҲгҖҒж—ҘгҖҒе°Ҹж—¶гҖҒеҲҶй’ҹгҖҒз§’гҖҒжҜ«з§’пјүгҖӮ

ж—Ҙжңҹи®ҫзҪ®ж–№жі•

и®ҫзҪ®ж–№жі•з”ЁдәҺи®ҫзҪ®ж—Ҙжңҹзҡ„жҹҗдёӘйғЁеҲҶгҖӮдёӢйқўжҳҜжңҖеёёз”Ёзҡ„ж–№жі•пјҲжҢүз…§еӯ—жҜҚйЎәеәҸжҺ’еәҸпјүпјҡ

ж–№жі• жҸҸиҝ°
setDate() д»Ҙж•°еҖјпјҲ1-31пјүи®ҫзҪ®ж—Ҙ
setFullYear() и®ҫзҪ®е№ҙпјҲеҸҜйҖүжңҲе’Ңж—Ҙпјү
setHours() и®ҫзҪ®е°Ҹж—¶пјҲ0-23пјү
setMilliseconds() и®ҫзҪ®жҜ«з§’пјҲ0-999пјү
setMinutes() и®ҫзҪ®еҲҶпјҲ0-59пјү
setMonth() и®ҫзҪ®жңҲпјҲ0-11пјү
setSeconds() и®ҫзҪ®з§’пјҲ0-59пјү
setTime() и®ҫзҪ®ж—¶й—ҙпјҲд»Һ 1970 е№ҙ 1 жңҲ 1 ж—ҘиҮід»Ҡзҡ„жҜ«з§’ж•°пјү

setFullYear() ж–№жі•

setFullYear() ж–№жі•и®ҫзҪ®ж—ҘжңҹеҜ№иұЎзҡ„е№ҙд»ҪгҖӮиҝҷдёӘдҫӢеӯҗи®ҫзҪ®дёә 2020 е№ҙпјҡ

е®һдҫӢ

<script>
var d = new Date();
d.setFullYear(2020);
document.getElementById("demo").innerHTML = d;
</script>

дәІиҮӘиҜ•дёҖиҜ•

е®һдҫӢ

<script>

дәІиҮӘиҜ•дёҖиҜ•

дәІиҮӘиҜ•дёҖиҜ•

someday = new Date();

if (someday > today) {
  text = "д»ҠеӨ©еңЁ 2049 е№ҙ 1 жңҲ 16 ж—Ҙд№ӢеүҚ";

  text = "д»ҠеӨ©еңЁ 2049 е№ҙ 1 жңҲ 16 ж—Ҙд№ӢеҗҺ";
