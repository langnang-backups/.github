Python дёӯзҡ„ж•°жҚ®зұ»еһӢ

й»ҳи®Өжғ…еҶөдёӢпјҢPython жӢҘжңүд»ҘдёӢж•°жҚ®зұ»еһӢпјҡ

  • strings - з”ЁдәҺиЎЁзӨәж–Үжң¬ж•°жҚ®пјҢж–Үжң¬з”Ёеј•еҸ·еј•иө·жқҘгҖӮдҫӢеҰӮ "ABCD"гҖӮ
  • integer - з”ЁдәҺиЎЁзӨәж•ҙж•°гҖӮдҫӢеҰӮ -1, -2, -3гҖӮ
  • float - з”ЁдәҺиЎЁзӨәе®һж•°гҖӮдҫӢеҰӮ 1.2, 42.42гҖӮ
  • boolean - з”ЁдәҺиЎЁзӨә True жҲ– FalseгҖӮ
  • complex - з”ЁдәҺиЎЁзӨәеӨҚе№ійқўдёӯзҡ„ж•°еӯ—гҖӮдҫӢеҰӮ 1.0 + 2.0jпјҢ1.5 + 2.5jгҖӮ

NumPy дёӯзҡ„ж•°жҚ®зұ»еһӢ

NumPy жңүдёҖдәӣйўқеӨ–зҡ„ж•°жҚ®зұ»еһӢпјҢе№¶йҖҡиҝҮдёҖдёӘеӯ—з¬Ұеј•з”Ёж•°жҚ®зұ»еһӢпјҢдҫӢеҰӮ i д»ЈиЎЁж•ҙж•°пјҢu д»ЈиЎЁж— з¬ҰеҸ·ж•ҙж•°зӯүгҖӮ

д»ҘдёӢжҳҜ NumPy дёӯжүҖжңүж•°жҚ®зұ»еһӢзҡ„еҲ—иЎЁд»ҘеҸҠз”ЁдәҺиЎЁзӨәе®ғд»¬зҡ„еӯ—з¬ҰгҖӮ

  • i - ж•ҙж•°
  • b - еёғе°”
  • u - ж— з¬ҰеҸ·ж•ҙж•°
  • f - жө®зӮ№
  • c - еӨҚеҗҲжө®зӮ№ж•°
  • m - timedelta
  • M - datetime
  • O - еҜ№иұЎ
  • S - еӯ—з¬ҰдёІ
  • U - unicode еӯ—з¬ҰдёІ
  • V - еӣәе®ҡзҡ„е…¶д»–зұ»еһӢзҡ„еҶ…еӯҳеқ— ( void )

жЈҖжҹҘж•°з»„зҡ„ж•°жҚ®зұ»еһӢ

NumPy ж•°з»„еҜ№иұЎжңүдёҖдёӘеҗҚдёә dtype зҡ„еұһжҖ§пјҢиҜҘеұһжҖ§иҝ”еӣһж•°з»„зҡ„ж•°жҚ®зұ»еһӢпјҡ

е®һдҫӢ

иҺ·еҸ–ж•°з»„еҜ№иұЎзҡ„ж•°жҚ®зұ»еһӢпјҡ

import numpy as np

arr = np.array([1, 2, 3, 4])

print(arr.dtype)

иҝҗиЎҢе®һдҫӢ

е®һдҫӢ

иҺ·еҸ–еҢ…еҗ«еӯ—з¬ҰдёІзҡ„ж•°з»„зҡ„ж•°жҚ®зұ»еһӢпјҡ

import numpy as np

arr = np.array(['apple', 'banana', 'cherry'])

print(arr.dtype)

иҝҗиЎҢе®һдҫӢ

з”Ёе·Іе®ҡд№үзҡ„ж•°жҚ®зұ»еһӢеҲӣе»әж•°з»„

жҲ‘д»¬дҪҝз”Ё array() еҮҪж•°жқҘеҲӣе»әж•°з»„пјҢиҜҘеҮҪж•°еҸҜд»ҘдҪҝз”ЁеҸҜйҖүеҸӮж•°пјҡdtypeпјҢе®ғе…Ғи®ёжҲ‘д»¬е®ҡд№үж•°з»„е…ғзҙ зҡ„йў„жңҹж•°жҚ®зұ»еһӢпјҡ

е®һдҫӢ

з”Ёж•°жҚ®зұ»еһӢеӯ—з¬ҰдёІеҲӣе»әж•°з»„пјҡ

import numpy as np

arr = np.array([1, 2, 3, 4], dtype='S')

print(arr)
print(arr.dtype)

иҝҗиЎҢе®һдҫӢ

еҜ№дәҺ iгҖҒuгҖҒfгҖҒS е’Ң UпјҢжҲ‘д»¬д№ҹеҸҜд»Ҙе®ҡд№үеӨ§е°ҸгҖӮ

е®һдҫӢ

еҲӣе»әж•°жҚ®зұ»еһӢдёә 4 еӯ—иҠӮж•ҙж•°зҡ„ж•°з»„пјҡ

import numpy as np

arr = np.array([1, 2, 3, 4], dtype='i4')

print(arr)
print(arr.dtype)

иҝҗиЎҢе®һдҫӢ

еҒҮеҰӮеҖјж— жі•иҪ¬жҚўдјҡжҖҺж ·пјҹ

еҰӮжһңз»ҷеҮәдәҶдёҚиғҪејәеҲ¶иҪ¬жҚўе…ғзҙ зҡ„зұ»еһӢпјҢеҲҷ NumPy е°Ҷеј•еҸ‘ ValueErrorгҖӮ

ValueErrorпјҡеңЁ Python дёӯпјҢеҰӮжһңдј йҖ’з»ҷеҮҪж•°зҡ„еҸӮж•°зҡ„зұ»еһӢжҳҜйқһйў„жңҹжҲ–й”ҷиҜҜзҡ„пјҢеҲҷдјҡеј•еҸ‘ ValueErrorгҖӮ

е®һдҫӢ

ж— жі•е°Ҷйқһж•ҙж•°еӯ—з¬ҰдёІпјҲжҜ”еҰӮ 'a'пјүиҪ¬жҚўдёәж•ҙж•°пјҲе°Ҷеј•еҸ‘й”ҷиҜҜпјүпјҡ

import numpy as np

arr = np.array(['a', '2', '3'], dtype='i')

иҝҗиЎҢе®һдҫӢ

иҪ¬жҚўе·Іжңүж•°з»„зҡ„ж•°жҚ®зұ»еһӢ

жӣҙж”№зҺ°жңүж•°з»„зҡ„ж•°жҚ®зұ»еһӢзҡ„жңҖдҪіж–№жі•пјҢжҳҜдҪҝз”Ё astype() ж–№жі•еӨҚеҲ¶иҜҘж•°з»„гҖӮ

astype() еҮҪж•°еҲӣе»әж•°з»„зҡ„еүҜжң¬пјҢе№¶е…Ғи®ёжӮЁе°Ҷж•°жҚ®зұ»еһӢжҢҮе®ҡдёәеҸӮж•°гҖӮ

ж•°жҚ®зұ»еһӢеҸҜд»ҘдҪҝз”Ёеӯ—з¬ҰдёІжҢҮе®ҡпјҢдҫӢеҰӮ 'f' иЎЁзӨәжө®зӮ№ж•°пјҢ'i' иЎЁзӨәж•ҙж•°зӯүгҖӮжҲ–иҖ…жӮЁд№ҹеҸҜд»ҘзӣҙжҺҘдҪҝз”Ёж•°жҚ®зұ»еһӢпјҢдҫӢеҰӮ float иЎЁзӨәжө®зӮ№ж•°пјҢint иЎЁзӨәж•ҙж•°гҖӮ

е®һдҫӢ

йҖҡиҝҮдҪҝз”Ё 'i' дҪңдёәеҸӮж•°еҖјпјҢе°Ҷж•°жҚ®зұ»еһӢд»Һжө®зӮ№ж•°жӣҙж”№дёәж•ҙж•°пјҡ

import numpy as np

arr = np.array([1.1, 2.1, 3.1])

newarr = arr.astype('i')

print(newarr)
print(newarr.dtype)

иҝҗиЎҢе®һдҫӢ

е®һдҫӢ

йҖҡиҝҮдҪҝз”Ё int дҪңдёәеҸӮж•°еҖјпјҢе°Ҷж•°жҚ®зұ»еһӢд»Һжө®зӮ№ж•°жӣҙж”№дёәж•ҙж•°пјҡ

import numpy as np

newarr = arr.astype(int)

print(newarr)
print(newarr.dtype)

иҝҗиЎҢе®һдҫӢ

е®һдҫӢ

е°Ҷж•°жҚ®зұ»еһӢд»Һж•ҙж•°жӣҙж”№дёәеёғе°”еҖјпјҡ
