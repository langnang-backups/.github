NumPy ufuncs

  • NumPy йҡҸжңә
  • е…Ҙй—Ё

д»Җд№ҲжҳҜ ufuncsпјҹ

ufuncs жҢҮзҡ„жҳҜвҖңйҖҡз”ЁеҮҪж•°вҖқпјҲUniversal FunctionsпјүпјҢе®ғд»¬жҳҜеҜ№ ndarray еҜ№иұЎиҝӣиЎҢж“ҚдҪңзҡ„ NumPy еҮҪж•°гҖӮ

дёәд»Җд№ҲиҰҒдҪҝз”Ё ufuncsпјҹ

ufunc з”ЁдәҺеңЁ NumPy дёӯе®һзҺ°зҹўйҮҸеҢ–пјҢиҝҷжҜ”иҝӯд»Је…ғзҙ иҰҒеҝ«еҫ—еӨҡгҖӮ

е®ғд»¬иҝҳжҸҗдҫӣе№ҝж’ӯе’Ңе…¶д»–ж–№жі•пјҢдҫӢеҰӮеҮҸе°‘гҖҒзҙҜеҠ зӯүпјҢе®ғд»¬еҜ№и®Ўз®—йқһеёёжңүеё®еҠ©гҖӮ

ufuncs иҝҳжҺҘеҸ—е…¶д»–еҸӮж•°пјҢжҜ”еҰӮпјҡ

where еёғе°”еҖјж•°з»„жҲ–жқЎд»¶пјҢз”ЁдәҺе®ҡд№үеә”еңЁдҪ•еӨ„иҝӣиЎҢж“ҚдҪңгҖӮ

dtype е®ҡд№үе…ғзҙ зҡ„иҝ”еӣһзұ»еһӢгҖӮ

out иҝ”еӣһеҖјеә”иў«еӨҚеҲ¶еҲ°зҡ„иҫ“еҮәж•°з»„гҖӮ

д»Җд№ҲжҳҜеҗ‘йҮҸеҢ–пјҹ

е°Ҷиҝӯд»ЈиҜӯеҸҘиҪ¬жҚўдёәеҹәдәҺеҗ‘йҮҸзҡ„ж“ҚдҪңз§°дёәеҗ‘йҮҸеҢ–гҖӮ

з”ұдәҺзҺ°д»Ј CPU е·Ій’ҲеҜ№жӯӨзұ»ж“ҚдҪңиҝӣиЎҢдәҶдјҳеҢ–пјҢеӣ жӯӨйҖҹеәҰжӣҙеҝ«гҖӮ

еҜ№дёӨдёӘеҲ—иЎЁзҡ„е…ғзҙ иҝӣиЎҢзӣёеҠ пјҡ

list 1: [1, 2, 3, 4]

list 2: [4, 5, 6, 7]

дёҖз§Қж–№жі•жҳҜйҒҚеҺҶдёӨдёӘеҲ—иЎЁпјҢз„¶еҗҺеҜ№жҜҸдёӘе…ғзҙ жұӮе’ҢгҖӮ

е®һдҫӢ

еҰӮжһңжІЎжңү ufuncпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ё Python зҡ„еҶ…зҪ® zip() ж–№жі•пјҡ

x = [1, 2, 3, 4]
y = [4, 5, 6, 7]
z = []

for i, j in zip(x, y):
  z.append(i + j)
print(z)

иҝҗиЎҢе®һдҫӢ

еҜ№жӯӨпјҢNumPy жңүдёҖдёӘ ufuncпјҢеҗҚдёә add(x, y)пјҢе®ғдјҡиҫ“еҮәзӣёеҗҢзҡ„з»“жһңгҖӮ

е®һдҫӢ

йҖҡиҝҮ ufuncпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ё add() еҮҪж•°пјҡ

import numpy as np

x = [1, 2, 3, 4]
y = [4, 5, 6, 7]
z = np.add(x, y)

print(z)
