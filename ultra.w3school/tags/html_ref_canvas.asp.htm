HTML Canvas еҸӮиҖғжүӢеҶҢ

HTML <canvas> ж Үзӯҫз”ЁдәҺйҖҡиҝҮи„ҡжң¬пјҲйҖҡеёёжҳҜ JavaScriptпјүеҠЁжҖҒз»ҳеҲ¶еӣҫеҪўгҖӮ

еҰӮйңҖдәҶи§Јжңүе…і <canvas> зҡ„жӣҙеӨҡдҝЎжҒҜпјҢиҜ·йҳ…иҜ»жҲ‘д»¬зҡ„ HTML Canvas ж•ҷзЁӢгҖӮ

йўңиүІгҖҒж ·ејҸе’ҢйҳҙеҪұ

еұһжҖ§ жҸҸиҝ°
fillStyle и®ҫзҪ®жҲ–иҝ”еӣһз”ЁдәҺеЎ«е……з»ҳз”»зҡ„йўңиүІгҖҒжёҗеҸҳжҲ–жЁЎејҸ
strokeStyle и®ҫзҪ®жҲ–иҝ”еӣһз”ЁдәҺз¬”и§Ұзҡ„йўңиүІгҖҒжёҗеҸҳжҲ–жЁЎејҸ
shadowColor и®ҫзҪ®жҲ–иҝ”еӣһз”ЁдәҺйҳҙеҪұзҡ„йўңиүІ
shadowBlur и®ҫзҪ®жҲ–иҝ”еӣһз”ЁдәҺйҳҙеҪұзҡ„жЁЎзіҠзә§еҲ«
shadowOffsetX и®ҫзҪ®жҲ–иҝ”еӣһйҳҙеҪұи·қеҪўзҠ¶зҡ„ж°ҙе№іи·қзҰ»
shadowOffsetY и®ҫзҪ®жҲ–иҝ”еӣһйҳҙеҪұи·қеҪўзҠ¶зҡ„еһӮзӣҙи·қзҰ»
ж–№жі• жҸҸиҝ°
createLinearGradient() еҲӣе»әзәҝжҖ§жёҗеҸҳпјҲз”ЁеңЁз”»еёғеҶ…е®№дёҠпјү
